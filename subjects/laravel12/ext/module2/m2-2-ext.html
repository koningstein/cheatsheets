Simpele Publieke Pagina

In de vorige module hebben we de lay-out voor het publieke deel gemaakt. Nu is het tijd om deze lay-out daadwerkelijk te gebruiken. We gaan de Homepagina en een simpele Over Ons pagina maken. Hierbij leren we hoe we data vanuit de route direct naar de view kunnen sturen.

Leeruitkomsten

  • Ik kan een controller aanmaken voor publieke pagina's.
  • Ik kan routes maken die verwijzen naar controller methodes.
  • Ik kan views maken die de publieke masterpage extenden.
  • Ik kan dynamische data (zoals paginatitels) meegeven aan een view.

De HomeController

Hoewel we routes direct in web.php kunnen definiÃ«ren met een closure functie, is het beter om een Controller te gebruiken. Dit houdt je routing bestand schoon en overzichtelijk (Separation of Concerns).

Maak een nieuwe controller aan:

php artisan make:controller HomeController

In deze controller maken we twee methodes: index (voor de homepagina) en about (voor de over-ons pagina).


namespace App\Http\Controllers;

use Illuminate\View\View;

class HomeController extends Controller
{
    public function index(): View
    {
        return view('home', [
            'title' => 'Welkom bij onze Webshop',
            'featured_products' => [] // Hier komt later echte data
        ]);
    }

    public function about(): View
    {
        return view('about', [
            'title' => 'Over Ons'
        ]);
    }
}

Routes Koppelen

Nu moeten we de URL's koppelen aan onze nieuwe controller. Open routes/web.php en voeg de routes toe.


use App\Http\Controllers\HomeController;
use Illuminate\Support\Facades\Route;

// Publieke routes
Route::get('/', [HomeController::class, 'index'])->name('home');
Route::get('/over-ons', [HomeController::class, 'about'])->name('about');

// De Admin routes die we eerder maakten staan hieronder (beveiligd)
// ...
Tip: Named Routes
Gebruik altijd ->name('...') bij je routes. Zo kun je in je navigatie (in navbarpublic.blade.php) verwijzen naar {{ route('about') }} in plaats van hardcoded /over-ons. Als je de URL later verandert naar /about-us, hoef je je navigatie niet aan te passen.
