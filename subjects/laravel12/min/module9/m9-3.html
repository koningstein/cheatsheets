Stacks & Push

Soms wil je vanuit een specifieke view (bijvoorbeeld een grafiek-pagina) een extra JavaScript bestand of CSS snippet toevoegen aan de <head> of footer van je layout.

Leeruitkomsten

  • Ik kan een @stack definiÃ«ren in de layout.
  • Ik kan content naar de stack sturen met @push.
  • Ik begrijp het verschil tussen @push en @prepend.

1. In de Layout (@stack)

Definieer een plek waar de scripts moeten landen.


<!-- layouts/app.blade.php -->
<head>
    <title>App</title>
    @vite(['resources/css/app.css'])

    {{-- Hier komen extra styles --}}
    @stack('styles')
</head>
<body>
    @yield('content')

    {{-- Hier komen extra scripts --}}
    @stack('scripts')
</body>

2. In de View (@push)

Duw de specifieke code naar de stack.


<!-- views/charts.blade.php -->
@extends('layouts.app')

@section('content')
    <div id="chart"></div>
@endsection

@push('scripts')
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
    <script>
        // Init chart...
    </script>
@endpush
