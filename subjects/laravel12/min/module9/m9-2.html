Components & Slots

Blade Components zijn d√© manier om herbruikbare stukken UI te bouwen. Denk aan knoppen, alerts of cards.

Leeruitkomsten

  • Ik kan een anoniem Blade component maken.
  • Ik kan content doorgeven via $slot.
  • Ik kan attributen doorgeven via $attributes.

1. Component Maken

Maak een bestand in resources/views/components. Bijvoorbeeld alert.blade.php.


<!-- resources/views/components/alert.blade.php -->
<div {{ $attributes->merge(['class' => 'p-4 rounded border']) }}>
    {{ $slot }}
</div>

2. Component Gebruiken

Je roept een component aan met de x- prefix.


<x-alert class="bg-red-100 border-red-400 text-red-700">
    Er is iets misgegaan!
</x-alert>

<x-alert class="bg-green-100 border-green-400 text-green-700">
    Succesvol opgeslagen!
</x-alert>

3. Props (Variabelen)

Je kunt ook specifieke variabelen doorgeven.


<!-- resources/views/components/button.blade.php -->
@props(['type' => 'submit'])

<button type="{{ $type }}" class="px-4 py-2 bg-blue-500 text-white rounded">
    {{ $slot }}
</button>
        

<!-- Gebruik -->
<x-button>Opslaan</x-button>
<x-button type="button">Annuleren</x-button>
