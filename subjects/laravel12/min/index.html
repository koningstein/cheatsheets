Laravel 12 Curriculum

Welkom bij de interactieve cheatsheet. Gebruik het menu aan de linkerkant om te navigeren door de installatie, het opdrachtentraject en de verdiepende thema's.

Opdrachten

Stap-voor-stap modules waarin we samen een webshop en admin-paneel bouwen.

Module 1

Basis CRUD

De basis van Laravel. Models, Migrations en Controllers opzetten voor een simpele administratie.

 Module 2

Users & Security

Authenticatie (inloggen/registreren), gebruikersbeheer en middleware voor beveiliging.

 Module 3

Advanced Admin

Een professioneel admin-dashboard bouwen met complexe formulieren, relaties en rechtenstructuren.

 Module 4

Webshop Features

De voorkant van de winkel. Producten tonen, winkelwagen functionaliteit en reviews systeem.

 Module 5

Orders & Validatie

Het afrekenproces, database transacties, complexe validatieregels en PDF facturen genereren.

Thema's

Verdiepende naslagwerken over specifieke onderdelen van het framework.

Database

Database & Models

Alles over database structuur, conventies, factories en geavanceerde Eloquent features.

 Relaties

Eloquent Relaties

Diepgang in hoe tabellen met elkaar communiceren (1-op-1, 1-op-veel, veel-op-veel).

 Routing

Routing & Controllers

Hoe URL's werken, dependency injection en slimme routing trucs.

 Frontend

Blade & Frontend

De view-laag. Components, layouts en integratie met frontend frameworks.

 Validatie

Validatie & Forms

Veilige input verwerking, custom error messages en form requests.

 Security

Security & Auth

Beveiliging, rechten (Gates/Policies), wachtwoorden hashen en e-mail verificatie.

 Features

Features

Extra functionaliteiten zoals bestandsopslag, externe API calls, events en mail.

 Automated Testing

Automated Testing

Code kwaliteit garanderen met tests (Pest, PHPUnit, Dusk).

 QA Tools & Workflow

QA Tools & Workflow

Professionele workflow met Static Analysis, Code Style tools en Git.
