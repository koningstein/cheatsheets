Layouts & Templates

We gaan een basis lay-out maken voor de admin-omgeving. Hiervoor gebruiken we een masterpage in Blade en Tailwind CSS voor de styling, zodat we een herbruikbare structuur hebben voor alle admin-pagina's.

Leeruitkomsten

  • Ik kan een Blade layout bestand maken.
  • Ik kan een Tailwind template aanpassen.
  • Ik kan een route naar een view koppelen.
LET OP VOOR DE OPDRACHTEN:
Laravel heeft tegenwoordig twee manieren voor layouts. Voor de opdrachten gebruiken we de KLASSIEKE methode (Deel 1).
De automatische tests zijn hierop gebaseerd. De moderne methode (Deel 2) staat hier ter info, omdat je die in de beroepspraktijk (en bij de Flux/Livewire starterkits) veel tegenkomt.

Deel 1: Klassieke Layouts (Voor de Opdrachten)

Dit is de basismanier van Laravel met @extends en @yield.

Stap 1: Bestand aanmaken

Maak het bestand: resources/views/layouts/layoutadmin.blade.php.

Stap 2: De Masterpage Code

Hier definiëren we de plekken waar content moet komen met @yield.


<!DOCTYPE html>
<html lang="nl">
<head>
    <meta charset="UTF-8">
    <title>Laravel Admin</title>
    <!-- Tailwind laden via Vite -->
    @vite(['resources/css/app.css', 'resources/js/app.js'])
</head>
<body class="bg-gray-100">

    <nav class="bg-white shadow p-4">
        <a href="/admin/categories" class="font-bold">Categorieën</a>
        <!-- Ruimte voor extra menu items -->
        @yield('topmenu')
    </nav>

    <main class="container mx-auto p-6">
        <!-- HIER komt de wisselende inhoud van subpagina's -->
        @yield('content')
    </main>

</body>
</html>

Stap 3: Gebruik in een View

In je pagina's (zoals index.blade.php) verwijs je naar deze layout:


@extends('layouts.layoutadmin')

@section('content')
    <h1>Pagina Inhoud</h1>
@endsection

Deel 2: Moderne Layouts (Livewire & Flux)

Dit gebruikt Components. Dit is de standaard in de nieuwe Livewire starterkit.

Flux Documentatie: https://fluxui.dev

Stap 1: Bestand Locatie (Anders!)

Het bestand moet staan in: resources/views/components/layouts/layoutadmin.blade.php.

Stap 2: De Component Code ($slot)

In plaats van @yield gebruiken we {{ $slot }}.


<!DOCTYPE html>
<html>
    <head>
        @vite(['resources/css/app.css', 'resources/js/app.js'])
        @fluxStyles
    </head>
    <body>
        <flux:sidebar>...</flux:sidebar>

        <flux:main>
            {{ $slot }} <!-- Hier komt de inhoud -->
        </flux:main>

        @fluxScripts
    </body>
</html>

Stap 3: Gebruik in een View

Je gebruikt een HTML-achtige tag die verwijst naar de component:


<x-layouts.layoutadmin>
    <h1>Pagina Inhoud</h1>
</x-layouts.layoutadmin>
