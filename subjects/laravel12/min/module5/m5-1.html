Stap 1: Models & Migrations Genereren

We hebben drie nieuwe entiteiten nodig: Order (de bestelling), Orderrow (de regels in de bestelling) en State (de status van de bestelling, bijv. 'betaald').


php artisan make:model Order -m
php artisan make:model Orderrow -m
php artisan make:model State -m

Stap 2: Migratie Orders

Een order hoort bij een gebruiker en heeft een status. We slaan ook de datum op.


// database/migrations/xxxx_xx_xx_create_orders_table.php
Schema::create('orders', function (Blueprint $table) {
    $table->id();
    $table->foreignId('user_id')->constrained(); // Wie bestelt?
    $table->foreignId('state_id')->constrained(); // Status (betaald/verzonden)
    $table->timestamps();
});

Stap 3: Migratie States

Simpele tabel voor statussen.


// database/migrations/xxxx_xx_xx_create_states_table.php
Schema::create('states', function (Blueprint $table) {
    $table->id();
    $table->string('name'); // Bijv. 'Pending', 'Paid', 'Shipped'
    $table->timestamps();
});

Stap 4: Migratie Orderrows

Hier slaan we op welk product is gekocht, hoeveel stuks, en - heel belangrijk - wat de prijs was op dat moment. Productprijzen kunnen namelijk veranderen, maar de prijs in een oude bestelling mag nooit meer wijzigen.


// database/migrations/xxxx_xx_xx_create_orderrows_table.php
Schema::create('orderrows', function (Blueprint $table) {
    $table->id();
    $table->foreignId('order_id')->constrained()->onDelete('cascade');
    $table->foreignId('product_id')->constrained()->onDelete('restrict'); // Product mag niet weg als het verkocht is.

    $table->integer('qty'); // Aantal
    $table->decimal('price', 8, 2); // Prijs op moment van koop
    $table->timestamps();
});

Stap 5: Relaties in Models

Vergeet niet de relaties in de Models te zetten, zodat we straks makkelijk $order->orderrows kunnen doen.


// app/Models/Order.php
public function user(): BelongsTo { return $this->belongsTo(User::class); }
public function state(): BelongsTo { return $this->belongsTo(State::class); }
public function orderrows(): HasMany { return $this->hasMany(Orderrow::class); }
        

// app/Models/Orderrow.php
public function product(): BelongsTo { return $this->belongsTo(Product::class); }
public function order(): BelongsTo { return $this->belongsTo(Order::class); }
