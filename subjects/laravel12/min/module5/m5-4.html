Orders Tonen

De klant moet zijn bestellingen kunnen inzien. We maken een index pagina voor de lijst en een show pagina voor de details.

Leeruitkomsten

  • Ik kan orders ophalen voor de ingelogde gebruiker (Auth::id()).
  • Ik kan orderregels en productinformatie ophalen via relaties (Eager Loading).

Stap 1: Controller Index

In de index methode tonen we alleen de bestellingen van de huidige gebruiker.


public function index()
{
    $orders = Order::where('user_id', Auth::id())
        ->latest()
        ->get();

    return view('open.orders.index', ['orders' => $orders]);
}

Stap 2: Index View

Maak een tabel in resources/views/open/orders/index.blade.php.


<table>
    @foreach($orders as $order)
        <tr>
            <td>#{{ $order->id }}</td>
            <td>{{ $order->created_at->format('d-m-Y') }}</td>
            <td>{{ $order->state->name }}</td>
            <td><a href="{{ route('open.orders.show', $order) }}">Bekijk</a></td>
        </tr>
    @endforeach
</table>

Stap 3: Controller Show

We moeten controleren of de order wel van de gebruiker is (Policy of check). En we moeten de regels + producten inladen.


public function show(Order $order)
{
    if ($order->user_id !== Auth::id()) {
        abort(403);
    }

    // Laad regels en producten
    $order->load('orderrows.product', 'state');

    return view('open.orders.show', ['order' => $order]);
}

Stap 4: Show View

Toon de orderregels in een tabel.


<h1>Bestelling #{{ $order->id }}</h1>
<p>Status: {{ $order->state->name }}</p>

<table>
    @foreach($order->orderrows as $row)
        <tr>
            <td>{{ $row->product->name }}</td>
            <td>{{ $row->qty }} x</td>
            <td>€ {{ $row->price }}</td>
        </tr>
    @endforeach
</table>

Stap 5: Routes


Route::middleware(['auth'])->group(function () {
    // ... store route stond er al ...
    Route::get('/orders', [OrderController::class, 'index'])->name('open.orders.index');
    Route::get('/orders/{order}', [OrderController::class, 'show'])
            ->name('open.orders.show');
});
