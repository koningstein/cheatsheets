Module 2-6: Controller Permissions

We hebben de "voordeur" beveiligd. Nu regelen we de regels binnenin: Sales mag kijken en wijzigen, maar niet verwijderen. Dit regelen we in de Controller.

Leeruitkomsten

Na het afronden van deze les kun je:

  • De HasMiddleware interface implementeren in een Controller.
  • Specifieke permissies koppelen aan meerdere acties (zowel de confirm-page als de delete-actie).
  • Knoppen in Blade verbergen met @can.

Stap 1: De Controller Aanpassen

Open app/Http/Controllers/Admin/CategoryController.php. We koppelen de permissies aan de methodes. 


namespace App\Http\Controllers\Admin;

use App\Http\Controllers\Controller;
use App\Models\Category;
use Illuminate\Http\Request;
// 1. Voeg deze imports toe:
use Illuminate\Routing\Controllers\HasMiddleware;
use Illuminate\Routing\Controllers\Middleware;
use Spatie\Permission\Middleware\PermissionMiddleware;

// 2. Voeg 'implements HasMiddleware' toe aan de class definitie
class CategoryController extends Controller implements HasMiddleware
{
    // 3. Voeg de static middleware functie toe
    public static function middleware(): array
    {
        return [
            // Koppel permissies aan methodes
            new Middleware(PermissionMiddleware::using('index category'), only:['index']),
            new Middleware(PermissionMiddleware::using('show category'), only:['show']),
            new Middleware(PermissionMiddleware::using('create category'), only:['create', 'store']),
            new Middleware(PermissionMiddleware::using('edit category'), only:['edit', 'update']),
            new Middleware(PermissionMiddleware::using('delete category'), only:['delete', 'destroy']),
        ];
    }

    // ... hieronder staan je gewone functies ...
}

Stap 2: De Delete Knop Verbergen

Nu kan een Sales-medewerker de delete-pagina niet meer zien. We verbergen de knop in het overzicht.

Open resources/views/admin/categories/index.blade.php.


@foreach($categories as $category)
    
        {{ $category->name }}
        
            @can('edit category')
                Aanpassen
            @endcan

            @can('delete category')
                 Verwijderen
            @endcan
        
    
@endforeach

Samenvatting

Je hebt nu permissies tot op het bot geregeld:

  • Route Middleware: Houdt klanten buiten de deur.
  • Controller Middleware: Zorgt dat Sales niet bij de delete-pagina (`delete`) en de verwijder-actie (`destroy`) kan.
  • Blade @can: Zorgt dat Sales de link niet ziet.
