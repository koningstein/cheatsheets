Simpele Publieke Pagina

We gaan een overzichtspagina maken voor de Categorieën die voor iedereen zichtbaar is. We scheiden de controllers in mappen: Admin en Open.

Leeruitkomsten

  • Ik kan controllers organiseren in submappen.
  • Ik kan data gepagineerd ophalen met Eloquent.
  • Ik kan paginatie-links weergeven in Blade.

Stap 1: Controller Aanmaken

Maak een controller in de map Open. We gebruiken geen resource controller omdat we alleen een index nodig hebben.

php artisan make:controller Open/CategoryController

Stap 2: Index Methode & Paginatie

Open app/Http/Controllers/Open/CategoryController.php. We gebruiken paginate() om de resultaten te verdelen over pagina's.


namespace App\Http\Controllers\Open;

use App\Http\Controllers\Controller;
use App\Models\Category;
use Illuminate\View\View;

class CategoryController extends Controller
{
    public function index(): View
    {
        // Haal 8 categorieën per pagina op
        $categories = Category::paginate(8);

        return view('open.categories.index', ['categories' => $categories]);
    }
}

Stap 3: Route & Alias

Om de code in web.php schoon te houden, gebruiken we een alias voor de namespace.


use App\Http\Controllers\Open as Open;

// Publieke route
Route::get('categories', [Open\CategoryController::class, 'index'])
    ->name('categories.index');

Stap 4: View met Paginatie Links

Maak resources/views/open/categories/index.blade.php. Gebruik $categories->links() om de navigatieknoppen te tonen.


@extends('layouts.layoutpublic')

@section('content')
    <h1 class="text-3xl mb-6">Kies een categorie</h1>

    <div class="grid grid-cols-1 md:grid-cols-4 gap-4">
        @foreach($categories as $category)
            <div class="border p-4 rounded shadow">
                <h2 class="font-bold text-xl">{{ $category->name }}</h2>
                <p>...</p>
            </div>
        @endforeach
    </div>

    <!-- Paginatie knoppen -->
    <div class="mt-8">
        {{ $categories->links() }}
    </div>
@endsection
