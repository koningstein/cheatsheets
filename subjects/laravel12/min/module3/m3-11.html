Delete View met Relaties

Wanneer we items verwijderen, is het belangrijk om de gebruiker te informeren over wat er precies verwijderd wordt, inclusief gerelateerde informatie. Dit voorkomt fouten.

Leeruitkomsten

  • Ik kan een delete view maken die gerelateerde informatie toont.
  • Ik kan disabled fields gebruiken voor dropdowns en inputs.

Stap 1: Controller (Delete)

We maken een aparte delete methode (naast destroy) om het bevestigingsscherm te tonen. We halen het product op via Model Binding.


public function delete(Product $product)
{
    return view('admin.products.delete', ['product' => $product]);
}

Stap 2: View: Disabled Fields

Het formulier lijkt op edit, maar alle velden zijn disabled. Dit maakt duidelijk dat de gebruiker hier niets kan wijzigen. Bij de categorie tonen we simpelweg de naam in een tekstveld in plaats van een select-menu, omdat je toch niets kunt kiezen.


<form action="{{ route('products.destroy', $product) }}" method="POST">
    @csrf
    @method('DELETE')

    <label class="block text-sm font-medium text-gray-700">Naam:</label>
    <input type="text" value="{{ $product->name }}" disabled class="mt-1 block w-full
            rounded-md border-gray-300 bg-gray-100 shadow-sm">

    <label class="block text-sm font-medium text-gray-700 mt-4">Categorie:</label>
    <!-- Gewoon de naam tonen, geen select nodig -->
    <input type="text" value="{{ $product->category->name }}" disabled class="mt-1
            block w-full rounded-md border-gray-300 bg-gray-100 shadow-sm">

    <label class="block text-sm font-medium text-gray-700 mt-4">Huidige Prijs:</label>
    <input type="text" value="{{ $product->latest_price->price }}" disabled class="mt-1
            block w-full rounded-md border-gray-300 bg-gray-100 shadow-sm">

    <div class="mt-6">
        <p class="text-red-600 mb-4">Weet je zeker dat je dit product wilt verwijderen?</p>
        <button type="submit" class="bg-red-600 text-white px-4 py-2 rounded
            hover:bg-red-700">Verwijderen</button>
        <a href="{{ route('products.index') }}" class="ml-4 text-gray-600
            hover:underline">Annuleren</a>
    </div>
</form>

Opdracht 26

Maak de Delete view voor Tasks. Zorg dat alle velden (inclusief Project en Activity) disabled zijn en de huidige waarde tonen.
