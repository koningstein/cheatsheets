Permissions in Controllers

Naast het beveiligen van routes met middleware, moeten we ook zorgen dat specifieke acties binnen controllers alleen uitgevoerd kunnen worden door gebruikers met de juiste permissies.

Leeruitkomsten

  • Ik kan permissies aanmaken en koppelen aan roles.
  • Ik kan permissies toepassen op controller methodes.
  • Ik kan routes beveiligen met permissies.
  • Ik kan authorize in requests op true zetten.

Stap 1: Permissies Aanmaken (Seeder)

Voordat we de controller kunnen beveiligen, moeten de permissies bestaan. We voegen deze toe in RoleAndPermissionSeeder.


// database/seeders/RoleAndPermissionSeeder.php

// 1. Maak permissies aan voor Product CRUD
Permission::create(['name' => 'index product']);
Permission::create(['name' => 'show product']);
Permission::create(['name' => 'create product']); // dekt create & store
Permission::create(['name' => 'edit product']);   // dekt edit & update
Permission::create(['name' => 'delete product']); // dekt delete & destroy

// 2. Koppel aan rol (bijv. admin mag alles)
$role = Role::findByName('admin');
$role->givePermissionTo(Permission::all());

Vergeet niet php artisan migrate:fresh --seed te draaien na wijzigingen!

Stap 2: Controller Setup

Open de ProductController. In Laravel 12 implementeren we de HasMiddleware interface om permissies te koppelen.


use Illuminate\Routing\Controllers\HasMiddleware;
use Illuminate\Routing\Controllers\Middleware;
use Spatie\Permission\Middleware\PermissionMiddleware;

// Voeg 'implements HasMiddleware' toe
class ProductController extends Controller implements HasMiddleware
{
    // ...
}

Stap 3: Middleware Methode

Voeg de statische middleware functie toe. Hier koppel je de permissie-naam aan de controller-acties.


public static function middleware(): array
{
    return [
        // Iedereen met 'index product' mag de lijst zien
        new Middleware(PermissionMiddleware::using('index product'), only:['index']),

        // Alleen met 'create product' mag je toevoegen
        new Middleware(PermissionMiddleware::using('create product'), only:['create', 'store']),

        // Enzovoort...
        new Middleware(PermissionMiddleware::using('edit product'), only:['edit', 'update']),
        new Middleware(PermissionMiddleware::using('delete product'), only:['delete', 'destroy']),
    ];
}
