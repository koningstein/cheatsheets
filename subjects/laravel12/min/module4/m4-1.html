Publieke Productpagina

In deze module maken we de klantkant van de webshop. We beginnen met een overzichtspagina van alle producten, zichtbaar voor iedereen.

Leeruitkomsten

  • Ik kan een publieke controller maken voor producten.
  • Ik kan producten ophalen met paginatie en eager loading.
  • Ik kan productgegevens tonen in een grid lay-out.

Stap 1: Controller Aanmaken

We scheiden de publieke logica van de admin. Maak een controller in de map Open.

php artisan make:controller Open/ProductController

Stap 2: Index Methode met Paginatie

Open app/Http/Controllers/Open/ProductController.php. We halen producten op, samen met de categorie en de laatste prijs om het N+1 probleem te voorkomen (zie Module 3).


namespace App\Http\Controllers\Open;

use App\Http\Controllers\Controller;
use App\Models\Product;
use Illuminate\View\View;

class ProductController extends Controller
{
    public function index(): View
    {
        // Haal producten op met relaties en paginatie (12 per pagina)
        $products = Product::with(['category', 'latest_price'])
            ->latest()
            ->paginate(12);

        return view('open.products.index', ['products' => $products]);
    }
}

Stap 3: Route Toevoegen

Voeg de route toe aan routes/web.php. Geef de route een naam voor makkelijk gebruik.


use App\Http\Controllers\Open\ProductController;

Route::get('/products', [ProductController::class, 'index'])
    ->name('open.products.index');

Stap 4: View Maken (Grid)

Maak resources/views/open/products/index.blade.php. Gebruik de publieke layout en een grid voor de weergave.


@extends('layouts.layoutpublic')

@section('content')
    <h1 class="text-3xl font-bold mb-6 text-center">Onze Producten</h1>

    <div class="grid grid-cols-1 md:grid-cols-3 lg:grid-cols-4 gap-6">
        @foreach($products as $product)
            <div class="border rounded-lg shadow-md overflow-hidden bg-white
            hover:shadow-xl transition"> <img src="https://placehold.co/400x300"
            alt="{{ $product->name }}" class="w-full h-48 object-cover">

                <div class="p-4">
                    <h2 class="font-bold text-xl mb-2">{{ $product->name }}</h2>
                    <p class="text-sm text-gray-500 mb-2">
                        {{ $product->category->name }}</p>

                    <p class="text-green-600 font-bold text-lg mb-2">
                        € {{ $product->latest_price->price ?? 'N/A' }}
                    </p>

                    <p class="text-gray-700 text-sm mb-4">
                        {{ Str::limit($product->description, 100) }}
                    </p>

                    <a href="{{ route('open.products.show', $product) }}" class="block
                    w-full bg-blue-600 text-white text-center py-2 rounded
                    hover:bg-blue-700">
                        Bekijken
                    </a>
                </div>
            </div>
        @endforeach
    </div>

    <div class="mt-8">
        {{ $products->links() }}
    </div>
@endsection
