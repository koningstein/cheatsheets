Winkelwagen Setup

Een webshop is niet compleet zonder winkelwagen. We gebruiken een bestaande package om dit makkelijk te implementeren.

Leeruitkomsten

  • Ik kan de Shoppingcart package installeren en configureren.
  • Ik kan items toevoegen aan de winkelwagen via een controller.

Stap 1: Package Installatie

We gebruiken de populaire bumbummen99/shoppingcart package. Installeer deze via Composer:

composer require bumbummen99/shoppingcart

Stap 2: Config Publiceren (Optioneel)

Als je instellingen wilt wijzigen (zoals belastingtarief), kun je de config publiceren:

php artisan vendor:publish
            --provider="Gloudemans\Shoppingcart\ShoppingcartServiceProvider" --tag="config"

Stap 3: CartController

Maak een CartController aan in de map Open met een store methode.


use Gloudemans\Shoppingcart\Facades\Cart;

public function store(Request $request, Product $product)
{
    // Cart::add(id, naam, aantal, prijs)
    Cart::add(
        $product->id,
        $product->name,
        1, // Aantal (standaard 1)
        $product->latest_price->price
    )->associate('App\Models\Product'); // Koppel aan model

    return redirect()->route('cart.index')
        ->with('status', 'Product toegevoegd!');
}

Stap 4: Route & Formulier

Maak de route aan in web.php:

Route::post('/cart/{product}', [CartController::class, 'store'])->name('cart.store');

Pas de "Bestel Nu" knop op show.blade.php aan zodat het een formulier wordt:


<form action="{{ route('cart.store', $product) }}" method="POST">
    @csrf
    <button type="submit" class="bg-green-600 text-white px-8 py-3 rounded-lg font-bold
            hover:bg-green-700 transition">
        Bestel Nu
    </button>
</form>
