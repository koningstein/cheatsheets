Reviews Setup (Database)

We gaan de mogelijkheid toevoegen om reviews achter te laten bij producten. Hiervoor moeten we eerst de database en models voorbereiden.

Leeruitkomsten

  • Ik kan een Model en Migratie maken voor Reviews.
  • Ik kan relaties definiëren tussen Review, User en Product.
  • Ik kan testdata genereren met een Factory en Seeder.

Stap 1: Model & Migratie Genereren

We gebruiken het artisan commando om in één keer het Model, de Migration, de Factory en de Seeder aan te maken.

php artisan make:model Review -mfs

Stap 2: Migratie Configureren

In de migration file definiëren we de tabelstructuur. Een review heeft een titel, inhoud, rating (score) en is gekoppeld aan een gebruiker en een product.


// database/migrations/xxxx_xx_xx_create_reviews_table.php
Schema::create('reviews', function (Blueprint $table) {
    $table->id();
    $table->string('title');
    $table->text('body');
    $table->integer('rating'); // Cijfer van 1 tot 5

    // Relaties met cascade delete (als user/product weg is, review ook weg)
    $table->foreignId('user_id')->constrained()->onDelete('cascade');
    $table->foreignId('product_id')->constrained()->onDelete('cascade');

    $table->timestamps();
});

Stap 3: Relaties in Models

Zorg dat Laravel de relaties snapt door ze in de Models te zetten.


// app/Models/Review.php
public function user() {
    return $this->belongsTo(User::class);
}
public function product() {
    return $this->belongsTo(Product::class);
}
        

// app/Models/Product.php
public function reviews() {
    return $this->hasMany(Review::class);
}

Stap 4: Factory & Seeding

Vul de `ReviewFactory.php` met neodata via Faker.


return [
    'title' => $this->faker->sentence,
    'body' => $this->faker->paragraph,
    'rating' => $this->faker->numberBetween(1, 5),
    'user_id' => User::all()->random()->id, // Kies een willekeurige user
    'product_id' => Product::all()->random()->id, // Kies een willekeurig product
];

Belangrijk: Voeg de seeder toe aan database/seeders/DatabaseSeeder.php zodat hij ook daadwerkelijk wordt uitgevoerd.


// DatabaseSeeder.php
public function run(): void
{
    $this->call([
        // ... andere seeders ...
        ReviewSeeder::class,
    ]);
}

Voer daarna het commando uit om alles te verversen:

php artisan migrate:fresh --seed
