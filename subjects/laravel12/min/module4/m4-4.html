Reviews Tonen

Nu we reviews in de database hebben, willen we deze ook tonen bij het product. Dit geeft klanten waardevolle informatie.

Leeruitkomsten

  • Ik kan reviews ophalen via Eager Loading.
  • Ik kan het gemiddelde van een collectie berekenen (avg).
  • Ik kan reviews weergeven op de productpagina.

Stap 1: Controller Uitbreiden

In de `show` methode van `ProductController` halen we nu ook de reviews op. We gebruiken `with('user')` om ook de naam van de schrijver te hebben, en `paginate(5)` zodat de pagina niet te lang wordt.


public function show(Product $product)
{
    // Haal de reviews op via de relatie, inclusief de gebruiker
    $reviews = $product->reviews()->with('user')->latest()->paginate(5);

    return view('open.products.show', [
        'product' => $product,
        'reviews' => $reviews
    ]);
}

Stap 2: Gemiddelde Score Berekenen

In de view kunnen we met de `avg()` methode op de reviews collectie direct het gemiddelde uitrekenen en tonen.


<div class="flex items-center mb-4">
    <span class="text-yellow-500 text-2xl mr-2">
        ★ {{ round($product->reviews->avg('rating'), 1) }}
    </span>
    <span class="text-gray-500">
        ({{ $product->reviews->count() }} reviews)
    </span>
</div>

Stap 3: Reviews Lijst

Voeg onderaan de `show.blade.php` een sectie toe voor de reviews.


<div class="mt-12">
    <h3 class="text-2xl font-bold mb-6">Klantbeoordelingen</h3>

    @foreach($reviews as $review)
        <div class="bg-gray-50 p-4 rounded-lg mb-4">
            <div class="flex justify-between items-start">
                <h4 class="font-bold text-lg">{{ $review->title }}</h4>
                <span class="text-yellow-500 font-bold">{{ $review->rating }} ★</span>
            </div>

            <p class="text-gray-700 mt-2">{{ $review->body }}</p>

            <p class="text-xs text-gray-400 mt-3">
                Geschreven door {{ $review->user->name }} -
                {{ $review->created_at->diffForHumans() }}
            </p>
        </div>
    @endforeach

    <div class="mt-4">
        {{ $reviews->links() }}
    </div>
</div>
