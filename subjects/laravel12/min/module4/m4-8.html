Winkelwagen Wijzigen

Gebruikers willen ook items kunnen verwijderen of het aantal aanpassen. Hiervoor voegen we acties toe aan de tabel.

Leeruitkomsten

  • Ik kan items in de winkelwagen updaten (Cart::update).
  • Ik kan items verwijderen (Cart::remove).
  • Ik kan formulieren gebruiken voor update en delete acties.

Stap 1: Routes

We hebben routes nodig voor update (PATCH) en delete (DELETE).


Route::patch('/cart/{rowId}', [CartController::class, 'update'])->name('cart.update');
Route::delete('/cart/{rowId}', [CartController::class, 'destroy'])->name('cart.destroy');

Stap 2: Wijzigen (View)

In de tabel (bij de acties kolom) plaatsen we een formulier om het aantal aan te passen.


<form action="{{ route('cart.update', $item->rowId) }}" method="POST"
            class="flex items-center gap-2">
    @csrf
    @method('PATCH')
    <input type="number" name="qty" value="{{ $item->qty }}" class="w-16 border
            rounded p-1" min="1">
    <button type="submit" class="text-blue-600 hover:underline">Update</button>
</form>

Stap 3: Verwijderen (View)

Daarnaast een knop (formulier) om het item volledig te verwijderen.


<form action="{{ route('cart.destroy', $item->rowId) }}" method="POST" class="mt-2">
    @csrf
    @method('DELETE')
    <button type="submit" class="text-red-600 hover:underline">Verwijder</button>
</form>

Stap 4: Controller Methodes

Voeg de update en destroy methodes toe aan CartController.


public function update(Request $request, $rowId)
{
    Cart::update($rowId, $request->qty); // Update het aantal
    return back()->with('status', 'Aantal aangepast!');
}

public function destroy($rowId)
{
    Cart::remove($rowId); // Verwijder item
    return back()->with('status', 'Product verwijderd!');
}
