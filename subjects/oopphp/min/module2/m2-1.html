Wat is autoloading?

Autoloading zorgt ervoor dat PHP automatisch de juiste class-bestanden laadt wanneer je een class gebruikt. Je hoeft dan niet meer handmatig require_once te schrijven voor elke class.

Het probleem zonder autoloading

In een Blackjack-applicatie heb je misschien classes zoals Card, Deck en Player. Zonder autoloading moet je elke class handmatig includen:

// Zonder autoloading - veel herhalende code
require_once 'Card.php';
require_once 'Deck.php';
require_once 'Player.php';

$card = new Card();
$deck = new Deck();
$player = new Player();

Dit wordt al snel onoverzichtelijk als je veel classes hebt!

Simpele autoload met spl_autoload_register()

Met spl_autoload_register() kun je een functie registreren. Deze functie wordt automatisch aangeroepen als PHP een class tegenkomt die nog niet is geladen.

// autoload.php
<?php
spl_autoload_register(function($className) {
    // Converteer class naam naar bestandsnaam
    $file = $className . '.php';

    // Controleer of bestand bestaat
    if (file_exists($file)) {
        require_once $file;
    }
});
?>

Hoe werkt het?

  1. Je roept spl_autoload_register() aan met een functie
  2. Als PHP een class tegenkomt die nog niet is geladen (zoals new Card()), wordt deze functie automatisch aangeroepen
  3. De functie krijgt de naam van de class als parameter ($className = "Card")
  4. De functie zoekt het juiste bestand (Card.php) en laadt dit in
  5. Nu kan PHP de class gebruiken

Gebruik in je applicatie

// index.php
<?php
require_once 'autoload.php';

// Classes worden nu automatisch geladen!
$card = new Card("Hearts", "Ace");
$deck = new Deck();
$player = new Player("John");
?>

Voordelen van autoloading

  • Minder code: je hebt geen lange lijst met require_once statements meer nodig
  • Automatisch: classes worden alleen geladen als je ze gebruikt (dit heet lazy loading)
  • Makkelijker onderhoud: nieuwe classes worden automatisch gevonden
  • Sneller: classes worden alleen geladen wanneer je ze nodig hebt
  • Overzichtelijker: je code blijft netjes georganiseerd
