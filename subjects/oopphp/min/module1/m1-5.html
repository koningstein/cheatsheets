Setters

Wat is een Setter?

Een setter is een speciale methode die je gebruikt om de waarde van een property te veranderen. Het grote voordeel van setters is dat je kunt controleren of de waarde klopt voordat je deze opslaat.

Voorbeeld: Kaartspel

Stel je voor: in een Blackjack-kaartspel wil je controleren of de punten van een kaart kloppen:

  • Punten kunnen niet negatief zijn
  • Punten kunnen niet hoger zijn dan 11 (Aas = 11, Plaatjes = 10)
  • Met een setter kun je deze controles uitvoeren voordat je de waarde opslaat

Waarom Setters gebruiken?

  • Validatie: Je kunt controleren of de waarde klopt voordat je deze opslaat
  • Veiligheid: Je voorkomt dat verkeerde gegevens worden opgeslagen
  • Terugkoppeling: Je kunt een bericht teruggeven of het gelukt is of niet

Een Setter maken

Een setter begint meestal met het woord set, gevolgd door de naam van de property:

class Card
{
    public $points;

    public function setPoints($points)
    {
        if ($points < 0) {
            return "Error: Points cannot be negative";
        }

        if ($points > 11) {
            return "Error: Points cannot exceed 11";
        }

        $this->points = $points;
        return "Points set successfully to " . $points;
    }
}

Een Setter gebruiken

Je roept een setter aan zoals elke andere methode, en je kunt het resultaat opvangen in een variabele:

$card = new Card();
$result = $card->setPoints(10);
echo $result;  // Output: Points set successfully to 10

$result = $card->setPoints(-5);
echo $result;  // Output: Error: Points cannot be negative

Tip

Laat setters altijd een bericht teruggeven, zodat je weet of het instellen van de waarde gelukt is of niet.
