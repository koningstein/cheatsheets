Constructor Promoted Properties

Wat zijn Constructor Promoted Properties?

Constructor Promoted Properties is een functie in PHP 8.0 die het maken van eigenschappen sneller maakt. Je kunt eigenschappen aanmaken én direct een waarde geven in één stap binnen de constructor.

Voorbeeld: Kaartspel

Bij het maken van een Card class moet je normaal:

  • Properties bovenaan de class declareren
  • Parameters in de constructor aanmaken
  • Elke property toewijzen met $this->property = $parameter;

Met Constructor Promoted Properties doe je dit alles in één keer!

Zonder Constructor Promoted Properties

De oude manier heeft drie aparte stappen nodig:

class Card
{
    public $suit;
    public $value;
    public $image;

    public function __construct($suit, $value, $image)
    {
        $this->suit = $suit;
        $this->value = $value;
        $this->image = $image;
    }
}

Met Constructor Promoted Properties

De nieuwe manier combineert alles in de constructor:

class Card
{
    public function __construct(
        public $suit,
        public $value,
        public $image
    ) {
        // Properties zijn automatisch gedeclareerd en toegewezen!
    }
}

Voordelen

  • Kortere code: Je hoeft eigenschappen niet apart aan te maken
  • Minder herhaling: Alles gebeurt op één plek
  • Overzichtelijker: Je ziet meteen welke eigenschappen een class heeft

Let op

Constructor Promoted Properties werken alleen voor properties die direct kunnen worden toegewezen zonder validatie. Voor properties die validatie nodig hebben, gebruik je setters apart.

Gebruik

Het aanmaken van objecten blijft hetzelfde:

$card = new Card("Hearts", "Ace", "ace_hearts.png");
