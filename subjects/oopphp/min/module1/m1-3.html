Properties

Wat zijn Properties?

Properties zijn de eigenschappen van een object. Ze bepalen welke informatie een object kan bevatten. Je kunt een property zien als een variabele die bij een class hoort. Elk object heeft zijn eigen waarden voor deze properties.

Voorbeeld: Kaartspel

Stel je voor: je maakt een Blackjack-kaartspel. Dan kun je een class Card maken met eigenschappen zoals:

  • suit: De soort kaart (bijvoorbeeld "Harten" of "Schoppen")
  • value: De waarde van de kaart (bijvoorbeeld "Koning" of "3")
  • points: De punten die de kaart waard is (bijvoorbeeld 10)
  • image: De bestandsnaam van de afbeelding die de kaart weergeeft

Properties toevoegen aan een Class

Je maakt een property door in de class het woord public te schrijven, gevolgd door de naam van de property:

class Card
{
    public $suit;
    public $value;
    public $points;
    public $image;
}

Properties gebruiken

Je kunt properties gebruiken en veranderen met de pijl-operator ->:

$card = new Card();
$card->suit = "Hearts";
$card->value = "Ace";
$card->points = 11;

echo $card->suit;  // Output: Hearts
