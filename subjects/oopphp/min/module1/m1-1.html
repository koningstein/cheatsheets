Class

Wat is een Class?

Een class is de blauwdruk van een object. Het is een sjabloon waarin je beschrijft wat voor eigenschappen en acties een object heeft. Je kunt een class vergelijken met een bouwplan voor een huis: het plan zelf is nog geen huis, maar je kunt er wel meerdere huizen mee bouwen.

Voorbeeld: Kaartspel

Stel je voor: je maakt een kaartspel. Dan kun je een class Card maken. Deze class beschrijft hoe een kaart eruitziet en wat een kaart kan doen. Maar de class is nog geen echte kaart! Pas later maak je echte kaarten zoals "Harten Aas" of "Schoppen Boer" op basis van deze class.

Hoe ziet een Class eruit?

Een class in PHP begint altijd met het woord class, gevolgd door de naam die je de class geeft. De opening curly bracket { zet je op de volgende regel.

class Card
{
    // Hier komt de inhoud
}

Belangrijke regels voor een Class

1. Naamgeving

Gebruik PascalCase voor de naam van een class:

  • Elk woord in de naam begint met een hoofdletter
  • Voorbeelden: Card, BlackjackPlayer, DeckManager

2. Bestandsnaam

De bestandsnaam moet precies hetzelfde zijn als de naam van de class, met .php erachter. Bijvoorbeeld: als je de class Card noemt, dan heet het bestand Card.php.


3. Structuur

De opening curly bracket { van een class zet je op de volgende regel, precies onder de naam van de class.

class Card
{
    // Properties en methods komen hier
}
class BlackjackPlayer
{
    // Properties en methods komen hier
}
