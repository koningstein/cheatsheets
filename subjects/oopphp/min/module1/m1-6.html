Getters

Wat is een Getter?

Een getter is een methode die je gebruikt om de waarde van een property op te halen. Getters geven je controle over hoe gegevens worden weergegeven en kunnen de gegevens aanpassen voordat je ze teruggeeft.

Voorbeeld: Kaartspel

Stel je voor: in een kaartspel wil je soms de waarde van een kaart op een bepaalde manier laten zien:

  • De waarde altijd met een hoofdletter laten beginnen (bijv. "ace" wordt "Ace")
  • De volledige naam van de kaart samenstellen uit meerdere properties
  • Berekende waarden teruggeven (bijv. totale punten van alle kaarten)

Waarom Getters gebruiken?

  • Controle: Je bepaalt hoe gegevens worden weergegeven
  • Aanpassen: Je kunt gegevens veranderen voordat je ze teruggeeft
  • Encapsulation (bescherming): Je verbergt de interne structuur van je object

Een Getter maken

Een getter begint meestal met het woord get, gevolgd door de naam van de property:

class Card
{
    public $value;

    public function getValue()
    {
        return ucfirst($this->value);  // Eerste letter hoofdletter
    }
}

Een Getter gebruiken

Je roept een getter aan zoals elke andere methode:

$card = new Card();
$card->value = "ace";
echo $card->getValue();  // Output: Ace

Getters en Setters combineren

In de praktijk gebruik je getters en setters vaak samen, zodat je volledige controle hebt over je properties:

class Card
{
    public $suit;
    public $value;

    public function setSuit($suit)
    {
        $validSuits = ["Hearts", "Diamonds", "Clubs", "Spades"];

        if (!in_array($suit, $validSuits)) {
            return "Error: Invalid suit";
        }

        $this->suit = $suit;
        return "Suit set to " . $suit;
    }

    public function getSuit()
    {
        return $this->suit;
    }
}
