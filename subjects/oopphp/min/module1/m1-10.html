Samenvatting: OOP Basis

Module 1 behandelde de basis van Object Oriented Programming in PHP. Je hebt geleerd hoe je classes en objects maakt, hoe je gegevens opslaat met properties, en hoe je functionaliteit toevoegt met methods.

Kernconcepten

Concept Uitleg Syntax (Voorbeeld)
Class Blauwdruk voor objecten class Card { }
Object Instantie van een class $card = new Card();
Property Eigenschap van een object public $suit;
Method Functie binnen een class public function display() { }
Setter Methode om property te wijzigen public function setSuit($suit) { }
Getter Methode om property op te halen public function getSuit() { }
Constructor Initialisatie bij object aanmaak public function __construct() { }
Promoted Properties Properties declareren in constructor __construct(public $suit) { }
Named Arguments Parameters bij naam aanroepen new Card(suit: "Hearts")

Belangrijke Keywords

Keyword Gebruik Voorbeeld
class Definieert een class class Card { }
new Maakt een nieuw object aan $card = new Card();
public Maakt property/method overal toegankelijk public $suit;
$this Verwijst naar huidige object $this->suit
-> Toegang tot property/method $card->suit
__construct Constructor methode naam public function __construct() { }

Best Practices

  • Naamgeving: Gebruik PascalCase voor class namen (bijv. Card, BlackjackPlayer)
  • Bestandsnaam: Moet precies hetzelfde zijn als de class naam (bijv. Card.php)
  • Constructor: Is altijd de eerste methode in een class
  • $this keyword: Schrijf $this->suit en NIET $this->$suit
  • Validatie: Gebruik setters voor properties die validatie nodig hebben
  • Promoted Properties: Gebruik alleen voor properties zonder validatie
