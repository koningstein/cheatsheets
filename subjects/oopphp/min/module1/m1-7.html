Constructor

Wat is een Constructor?

Een constructor is een speciale methode die automatisch wordt uitgevoerd wanneer je een nieuw object aanmaakt. Je gebruikt een constructor om eigenschappen van een object direct in te stellen bij het aanmaken.

Voorbeeld: Kaartspel

Stel je voor: bij het aanmaken van een kaart wil je direct de belangrijkste eigenschappen instellen:

  • Zonder constructor: je moet na het aanmaken elke property apart instellen
  • Met constructor: alle properties worden in één keer ingesteld bij het aanmaken
  • Dit voorkomt dat je een "lege" kaart hebt zonder waarden

Waarom een Constructor gebruiken?

  • Efficiënt: Je bespaart tijd door alle eigenschappen in één keer in te stellen
  • Consistent: Alle objecten worden op dezelfde manier geïnitialiseerd
  • Flexibel: Je kunt eenvoudig meerdere objecten maken met verschillende beginwaarden

Een Constructor maken

In PHP heet een constructor altijd __construct. Je kunt parameters toevoegen om waarden aan properties toe te wijzen:

Tip

De constructor zet je altijd als eerste methode in een class. Dit maakt de code overzichtelijk en laat meteen zien welke waarden nodig zijn om een object aan te maken.

class Card
{
    public $suit;
    public $value;
    public $image;

    public function __construct($suit, $value, $image)
    {
        $this->suit = $suit;
        $this->value = $value;
        $this->image = $image;
    }
}

Een Constructor gebruiken

Wanneer je een object aanmaakt met new, geef je de waarden automatisch door aan de constructor:

$card = new Card("Hearts", "King", "king_hearts.png");

echo $card->suit;   // Output: Hearts
echo $card->value;  // Output: King

Begrippen

  • Parameters: De variabelen in de constructor, zoals $suit en $value
  • Argumenten: De echte waarden die je meegeeft, zoals "Hearts" en "King"
