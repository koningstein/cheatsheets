Methoden

Wat zijn Methoden?

Methoden zijn functies die bij een class horen. Ze beschrijven wat een object kan doen. Terwijl properties de gegevens van een object zijn, bepalen methoden de acties.

Voorbeeld: Kaartspel

Een Card class kan bijvoorbeeld deze methoden hebben:

  • display(): Toont de informatie van de kaart
  • flip(): Draait de kaart om (van open naar dicht of andersom)
  • getValue(): Geeft de waarde van de kaart terug

Methoden toevoegen

Je maakt een methode binnen een class op dezelfde manier als een functie, maar dan binnen de curly brackets van de class:

class Card
{
    public $suit;
    public $value;
    public $points;

    public function display()
    {
        echo "This is the " . $this->value . " of " . $this->suit;
    }
}

Je kunt ook de kortere schrijfwijze gebruiken met curly brackets binnen een string:

public function display()
{
    echo "This is the {$this->value} of {$this->suit}";
}

Het $this keyword

Het keyword $this verwijst naar het huidige object. Je gebruikt $this om binnen een methode toegang te krijgen tot de properties van het object.

Methoden aanroepen

Buiten de class roep je een methode aan met de pijl-operator ->:

$card = new Card();
$card->suit = "Hearts";
$card->value = "King";
$card->display();  // Output: This is the King of Hearts

Let op

Gebruik altijd $this-> binnen een methode om naar properties van het huidige object te verwijzen. Vergeet de $ niet voor this!

Let op: schrijf $this->suit en NIET $this->$suit. De naam van de property krijgt zelf geen $ symbool!
