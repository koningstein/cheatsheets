Object

Wat is een Object?

Een object is een instantie van een class. Met instantie bedoelen we: een concreet voorbeeld. Als een class de blauwdruk is, dan is een object het echte ding dat je op basis van die blauwdruk maakt. Je kunt van één class meerdere verschillende objecten maken.

Vergelijking

  • Class: De blauwdruk voor een kaart
  • Object: Een echte kaart, zoals "Harten Aas" of "Schoppen Koning"

Een Object maken

Je maakt een object met het woord new, gevolgd door de naam van de class en ronde haakjes ().

$cardAceHearts = new Card();
$cardKingSpades = new Card();

Beide objecten zijn gemaakt met de class Card, maar ze zijn compleet onafhankelijk van elkaar. Ze hebben allebei hun eigen gegevens.

Objecten uitprinten

Je kunt een object niet met echo weergeven. Gebruik var_dump() om te kijken wat er in een object zit:

var_dump($cardAceHearts);

Tip

Gebruik <pre> tags om de output van var_dump() netjes en overzichtelijk te zien.

Waarom gebruik je Objecten?

  • Flexibel: Je kunt zoveel objecten maken als je wilt van dezelfde class
  • Overzichtelijk: Elk object heeft zijn eigen gegevens die niet door elkaar lopen
  • Efficiënt: Je schrijft de class één keer en kunt er daarna oneindig veel objecten mee maken
