Objecten als datatype

Wat betekent objecten als datatype?

Je kunt objecten (instanties van classes) gebruiken als datatype voor properties en parameters. Dit maakt duidelijk hoe classes met elkaar verbonden zijn en zorgt voor veiligere code.

Waarom objecten als datatype?

  • Relaties duidelijk maken: Een Deck bevat Card objecten
  • Veiligheid: Je kunt alleen objecten van het juiste type toevoegen
  • Betere code organisatie: Logische groepering van functionaliteit
  • Hulp van je programma: Autocomplete werkt perfect

Object als parameter

Gebruik de Classnaam als type hint. Dit zorgt ervoor dat je alleen een object van die specifieke class kunt doorgeven:

public function addCard(Card $card): void
{
    $this->cards[] = $card;
}

public function dealCard(Player $player): void
{
    $card = array_shift($this->cards);
    $player->addCard($card);
}

Object als property type

Door een property een classnaam als type te geven, zorg je ervoor dat deze property alleen een object van die class kan bevatten:

class Game
{
    private Deck $deck;
    private Player $player;
    private Player $dealer;

    public function __construct(Player $player, Player $dealer)
    {
        $this->player = $player;
        $this->dealer = $dealer;
        $this->deck = new Deck();
    }
}

Nullable object types

Soms wil je dat een property een object kan bevatten, maar ook null mag zijn. Gebruik dan een vraagteken (?) voor de classnaam:

class Game
{
    private ?Player $winner = null;

    public function setWinner(?Player $winner): void
    {
        $this->winner = $winner;
    }

    public function getWinner(): ?Player
    {
        return $this->winner;
    }
}

// Gebruik:
$game = new Game();
$game->setWinner($player1); // OK - er is een winnaar
$game->setWinner(null); // Ook OK - het spel is nog niet afgelopen

Array van objecten

Je kunt in de code alleen array als type gebruiken. Maar met een Docblock maak je duidelijk dat de array alleen objecten van een bepaalde class bevat:

class Player
{
    private array $hand = []; // Array of Card objects

    public function addCard(Card $card): void
    {
        $this->hand[] = $card;
    }

    /**
     * @return Card[] Array of Card objects
     */
    public function getHand(): array
    {
        return $this->hand;
    }
}

Best practice

Gebruik altijd object types in plaats van arrays met verschillende soorten inhoud. Dit maakt je code betrouwbaarder:

// ❌ Niet doen:
private array $hand; // Wat zit er in deze array?

// ✅ Wel doen (met docblock):
private array $hand = [];

/**
 * @return Card[] Array of Card objects
 */
public function getHand(): array
{
    return $this->hand;
}
