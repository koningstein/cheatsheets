Code kwaliteit met PHPStan

PHPStan is een statische analyse tool. Het scant je code op fouten (zoals verkeerde types of vergeten checks) zonder de code uit te voeren.

1. Installatie & Configuratie

Installeer PHPStan als 'dev-tool' (alleen voor tijdens het bouwen):

composer require --dev phpstan/phpstan

Maak in je hoofdmap een bestand genaamd phpstan.neon:

parameters:
    level: 6          # Wij adviseren level 6 voor goede type-checking
    paths:
        - src         # Welke map scannen?

2. Uitvoeren

Run de check via je terminal:

vendor/bin/phpstan analyse

3. Veelvoorkomende Fouten

Foutmelding Betekenis & Oplossing
Method has no return type specified Je bent vergeten: : int of : void achter je functie te zetten.
Property has no type specified Je bent vergeten: private string $name te typen (het type ontbreekt).
Call to method on possibly null Gevaarlijk: Je probeert iets te doen met een object dat misschien niet bestaat.
Oplossing: Voeg een check toe: if ($speler !== null) { ... }

4. Levels

Je kunt instellen hoe streng PHPStan is (0 t/m 9).

  • Level 0-3: Basis checks (bestaat deze functie wel?).
  • Level 4-5: Type checks (stop je geen tekst in een getal?).
  • Level 6: Controleert ook op ontbrekende typehints (Aanbevolen).
  • Level 9: Extreem streng (ook gemengde types en 'strict' checks).
