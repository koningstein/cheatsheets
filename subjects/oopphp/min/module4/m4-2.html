Smarty Template Engine

Smarty is een template engine die PHP-code (logica) scheidt van HTML (presentatie). Dit houdt je code overzichtelijk en veilig.

1. Installatie & Structuur

Installeer Smarty via Composer:

composer require smarty/smarty

Hanteer deze mappenstructuur:

project/
├── templates/          # .tpl bestanden (HTML)
├── templates_c/        # Cache (automatisch gevuld)
├── src/                # PHP Classes
└── index.php           # PHP Logica

Belangrijk

Voeg templates_c/ toe aan je .gitignore. Deze map bevat tijdelijke bestanden.

2. Configureren (index.php)

require 'vendor/autoload.php';
use Smarty\Smarty;

// Smarty initialiseren
$smarty = new Smarty();
$smarty->setTemplateDir('templates');
$smarty->setCompileDir('templates_c');

// Data klaarzetten
$player = new Player("Sjaak");
$smarty->assign('speler', $player);
$smarty->assign('score', 21);

// Template tonen
$smarty->display('game.tpl');

3. Syntax (.tpl)

In de templates map maak je HTML-bestanden met .tpl extensie. Smarty gebruikt accolades {} voor logica.

Variabelen tonen

<!-- Een variabele -->
<p>Score: {$score}</p>

<!-- Een object methode -->
<p>Naam: {$speler->getName()}</p>

Keuzes maken (If/Else)

{if $score > 21}
    <p>Helaas, je bent af (Bust)!</p>
{elseif $score == 21}
    <p>Blackjack!</p>
{else}
    <p>Wil je nog een kaart?</p>
{/if}

Lijsten doorlopen (Foreach)

<ul>
{foreach $cards as $card}
    <li>{$card->getRank()} van {$card->getSuit()}</li>
{/foreach}
</ul>
