Layout met Bootstrap

We gebruiken Bootstrap voor de opmaak en Smarty Inheritance om een vaste layout (zoals een menu en footer) te maken die op elke pagina terugkomt.

1. Bestanden & Mappen

Download de Bootstrap bestanden en zet ze in je project:

templates/
├── css/
│   ├── bootstrap.min.css
│   └── sticky-footer-navbar.css
├── js/
│   └── bootstrap.bundle.min.js
└── layout.tpl   (Je basis bestand)

2. De Basis (layout.tpl)

Dit bestand bevat de HTML die op elke pagina hetzelfde is (head, menu, footer). Op de plek waar de inhoud moet komen, gebruik je een block.

<!doctype html>
<html>
<head>
    <!-- Bootstrap CSS koppelen -->
    <link href="templates/css/bootstrap.min.css" rel="stylesheet">
</head>
<body class="d-flex flex-column h-100">

    <!-- Hier komt je Menu -->
    <nav>...</nav>

    <!-- HIER komt de wisselende inhoud -->
    <main class="container">
        {block name="content"}{/block}
    </main>

    <!-- Hier komt je Footer -->
    <footer>...</footer>

</body>
</html>

3. De Pagina (game.tpl)

In je losse pagina's gebruik je extends om de basis te laden. Alles wat je in het block zet, wordt in de layout "geïnjecteerd".

{extends file="layout.tpl"}

{block name="content"}
    <h1>Mijn Blackjack Spel</h1>

    <div class="alert alert-success">
        Welkom {$speler->getName()}!
    </div>
{/block}

4. Bootstrap Cheat Sheet

Veelgebruikte classes om je spel snel mooi te maken:

Doel Classes
Kleuren bg-success (Groen)
bg-danger (Rood)
bg-primary (Blauw)
bg-dark (Zwart)
Ruimte (Spacing) mt-3 (Margin Top)
mb-3 (Margin Bottom)
p-3 (Padding rondom)
Componenten btn btn-primary (Knop)
alert alert-info (Melding)
card (Kader met rand)
Indeling (Grid) row (Rij)
col-6 (Halve breedte)
