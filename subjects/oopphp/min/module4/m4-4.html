Formulieren en OOP

Formulieren laten gebruikers data invoeren. In plaats van hardcoded objecten, gebruiken we deze data om flexibele objecten te maken.

1. Het Formulier (Smarty Template)

In een .tpl bestand maak je het formulier. Let op de attributen van de <form> en <input> tags.

<!-- templates/form.tpl -->
{extends file="layout.tpl"}

{block name="content"}
    <!-- action: Waar gaat de data heen? -->
    <!-- method: Hoe? (POST voor data) -->
    <form action="index.php?page=create" method="POST">

        <label>Naam:</label>
        <!-- name: Dit wordt de key in $_POST['name'] -->
        <input type="text" name="name" class="form-control" required>

        <button type="submit" class="btn btn-success">Opslaan</button>
    </form>
{/block}

2. Routing (De wegwijzer)

We gebruiken één bestand (index.php) om alles te regelen. Met $_GET['page'] bepalen we wat we laten zien.

// index.php

// Kijk in de URL (?page=...) of er een pagina is gekozen.
// Zo niet (??), gebruik standaard 'home'.
$page = $_GET['page'] ?? 'home';

// Routing Switch
if ($page === 'create') {
    // Toon formulier
    $template->display('form.tpl');
}
elseif ($page === 'home') {
    // Toon homepagina
    $template->display('home.tpl');
}

3. Belangrijke termen

Term Betekenis
action="..." Bepaalt naar welke URL het formulier wordt gestuurd.
method="POST" Verstuurt data op de achtergrond (veilig). Data komt in $_POST.
name="..." De naam van het veld. Cruciaal voor PHP: <input name="stad"> wordt $_POST['stad'].
?? (Null Coalescing) Korte manier om een standaardwaarde te geven als een variabele niet bestaat.
