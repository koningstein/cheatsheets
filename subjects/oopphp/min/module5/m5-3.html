Inheritance Basis (Extends)

Het Extends Keyword

Om code duplicatie te voorkomen, gebruiken we Inheritance (overerving). We definiëren een algemene "ouder" (Parent) class met alle gedeelde code, en laten de specifieke classes (Children) daarvan "erven".

In PHP doe je dit met het keyword extends.

Stap 1: De Parent Class (Player.php)

Eerst schonen we de Player class op. Hierin laten we alleen de code staan die voor iedereen geldt (zowel mensen als dealers).

class Player
{
    // Iedereen heeft kaarten nodig
    protected array $hand = [];

    // Iedereen moet kaarten kunnen krijgen
    public function addCard(Card $card)
    {
        $this->hand[] = $card;
    }

    // Iedereen heeft een score nodig
    public function getScore(): int
    {
        // ... logica om score te berekenen ...
    }
}

Stap 2: De Child Classes (Dealer.php)

Nu kunnen we de Dealer class heel simpel maken. We hoeven de methodes addCard en getScore niet meer te typen, want die "erft" hij van Player.

// Let op: extends Player
class Dealer extends Player
{
    // Deze class is nu leeg!
    // Maar... hij heeft wél addCard() en getScore() tot zijn beschikking.
}

"Is een..." relatie

Door inheritance te gebruiken, zeggen we: "Een Dealer is een Player". Alles wat een Player kan, kan een Dealer nu ook.

Stap 3: Specifieke Child Classes (HumanPlayer.php)

De HumanPlayer erft ook van Player, maar voegt zijn eigen unieke eigenschappen toe (geld).

class HumanPlayer extends Player
{
    // Unieke eigenschap voor HumanPlayer
    private int $balance = 1000;

    // Unieke methode voor HumanPlayer
    public function placeBet(int $amount)
    {
        $this->balance -= $amount;
    }
}

Het resultaat

Je kunt nu gewoon methodes aanroepen op de Dealer, ook al staan ze niet letterlijk in zijn bestand:

$dealer = new Dealer();
$dealer->addCard(new Card('Hearts', 'Ace')); // Werkt via overerving!
echo $dealer->getScore(); // Werkt ook!

Terminologie

  • Parent Class (of Super Class): De basis class (Player).
  • Child Class (of Sub Class): De class die erft (Dealer).
  • Inheritance: Het mechanisme waarmee properties en methods worden doorgegeven.
