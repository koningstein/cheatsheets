Layout met Bootstrap - Sticky Footer Navbar

Wat is Bootstrap?

Bootstrap is een populair front-end framework waarmee je snel professionele en responsive websites kunt bouwen. Het biedt:

  • Voorgedefinieerde CSS-classes voor typografie, knoppen, formulieren, tabellen, etc.
  • Responsive grid systeem voor verschillende schermformaten
  • JavaScript componenten zoals modals, dropdowns en carousels
  • Consistente styling zonder zelf veel CSS te schrijven

Sticky Footer Navbar

De Sticky Footer Navbar is een veelgebruikt layout patroon waarbij:

  • De navigatiebalk bovenaan blijft plakken tijdens scrollen
  • De footer altijd onderaan staat, ook bij weinig content
  • De main content zich tussen header en footer bevindt

Stap 1: Bootstrap bestanden lokaal opslaan

Eerst maken we de benodigde mappenstructuur en downloaden we de Bootstrap bestanden:

Mappenstructuur aanmaken

templates/
├── css/
│   ├── bootstrap.min.css
│   └── sticky-footer-navbar.css
└── js/
    ├── color-modes.js
    └── bootstrap.bundle.min.js

Let op

Ga naar de Bootstrap Sticky Footer Navbar voorbeeld en bekijk de broncode om de bestanden te kopiëren.

Stap 2: Layout template maken

We maken een layout.tpl bestand dat als basis dient voor alle pagina's. Deze layout bevat de HTML structuur, Bootstrap bestanden en een Smarty block voor de content.

templates/layout.tpl

<!doctype html>
<html lang="nl">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Blackjack Game</title>

    <!-- Bootstrap CSS -->
    <link href="./templates/css/bootstrap.min.css" rel="stylesheet">
    <link href="./templates/css/sticky-footer-navbar.css" rel="stylesheet">
</head>
<body class="d-flex flex-column h-100">

    <!-- Navigatie -->
    <nav class="navbar navbar-expand-md navbar-dark bg-dark">
        <div class="container-fluid">
            <a class="navbar-brand" href="index.php">Blackjack</a>
            <button class="navbar-toggler" type="button"
                    data-bs-toggle="collapse"
                    data-bs-target="#navbarCollapse">
                <span class="navbar-toggler-icon"></span>
            </button>
            <div class="collapse navbar-collapse" id="navbarCollapse">
                <ul class="navbar-nav me-auto mb-2 mb-md-0">
                    <li class="nav-item">
                        <a class="nav-link" href="index.php">Home</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="index.php?action=newGame">Nieuw Spel</a>
                    </li>
                </ul>
            </div>
        </div>
    </nav>

    <!-- Main content -->
    <main class="flex-shrink-0">
        <div class="container">
            {block name="content"}{/block}
        </div>
    </main>

    <!-- Footer -->
    <footer class="footer mt-auto py-3 bg-body-tertiary">
        <div class="container">
            <span class="text-muted">Blackjack Game &copy; 2025</span>
        </div>
    </footer>

    <!-- Bootstrap JS -->
    <script src="./templates/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Let op de volgende onderdelen:

  • d-flex flex-column h-100 op body zorgt dat de footer onderaan blijft
  • {block name="content"}{/block} is de plek waar child templates hun content plaatsen
  • Lokale paden naar CSS en JS bestanden (let op: geen integrity en crossorigin attributen)

Stap 3: Template aanpassen om layout te gebruiken

Nu passen we een bestaand template aan om de layout te gebruiken. We gebruiken {extends} om de layout te laden en {block} om de content te vullen.

Voorbeeld 1: Game overzicht (templates/gameOverview.tpl)

{extends file="layout.tpl"}

{block name="content"}
    <h1 class="mt-5">Blackjack Game</h1>

    <div class="row mt-4">
        <div class="col-md-6">
            <div class="card">
                <div class="card-header bg-danger text-white">
                    <h3>Dealer</h3>
                </div>
                <div class="card-body">
                    <p><strong>Kaarten:</strong> {$dealer->getCardsAsString()}</p>
                    <p><strong>Score:</strong> {$dealer->getScore()}</p>
                </div>
            </div>
        </div>

        <div class="col-md-6">
            <div class="card">
                <div class="card-header bg-success text-white">
                    <h3>Speler</h3>
                </div>
                <div class="card-body">
                    <p><strong>Kaarten:</strong> {$player->getCardsAsString()}</p>
                    <p><strong>Score:</strong> {$player->getScore()}</p>
                </div>
            </div>
        </div>
    </div>

    <div class="alert alert-info mt-4" role="alert">
        <h4 class="alert-heading">Resultaat</h4>
        <p>{$gameResult}</p>
    </div>
{/block}

Bootstrap componenten gebruikt:

  • .card - Voor het weergeven van dealer en speler informatie
  • .card-header en .card-body - Voor structuur binnen de card
  • .bg-danger (rood) voor dealer, .bg-success (groen) voor speler
  • .row en .col-md-6 - Voor responsive grid layout (2 kolommen)
  • .alert - Voor het tonen van het spelresultaat
  • .mt-4, .mt-5 - Margin-top utility classes voor spacing

Bootstrap componenten gebruikt:

  • .table, .table-striped, .table-hover - Voor gestylede tabellen
  • .table-dark - Donkere header
  • .badge - Voor gekleurde score indicators
  • .btn, .btn-primary - Voor knoppen

Stap 4: Index.php aanpassen

De index.php hoeft niet aangepast te worden als je al Smarty hebt geïnitialiseerd. De templates gebruiken automatisch de layout door de {extends} directive.

<?php
require_once 'vendor/autoload.php';

use Smarty\Smarty;

// Smarty initialiseren
$template = new Smarty();
$template->setTemplateDir('./templates/');

// Game logica
$game = new BlackjackGame();
$game->deal();
$result = $game->determineWinner();

// Gegevens toewijzen aan template
$template->assign('dealer', $game->getDealer());
$template->assign('player', $game->getPlayer());
$template->assign('gameResult', $result);

// Template weergeven (gebruikt automatisch layout.tpl)
$template->display('gameOverview.tpl');

Belangrijke punten:

  • Gebruik lokale bestanden in plaats van CDN voor Bootstrap
  • Verwijder integrity en crossorigin attributen bij lokale bestanden
  • Gebruik {extends file="layout.tpl"} aan het begin van elk child template
  • Plaats alle content binnen {block name="content"}...{/block}
  • Bootstrap classes werken automatisch zonder extra configuratie
