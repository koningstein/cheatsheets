Getters: Data Formatteren

In de vorige lessen hebben we properties gemaakt waar je data in kunt stoppen (Set) en uit kunt halen (Get). Vaak is die data "rauw".

Bijvoorbeeld: Energy is gewoon het getal 50. Maar misschien wil je dit op het scherm tonen als een mooie tekst: "ðŸ”‹ Energie: 50 / 100".

Je kunt dit steeds in Program.cs aan elkaar plakken, maar het is veel slimmer om dit in de Class zelf te regelen met een speciale Getter Methode.

Wat is een Getter Methode?

Een getter is een methode die niets anders doet dan data ophalen, het mooi maken (formatteren) en teruggeven. Hij verandert niets aan het object, hij "leest" alleen.

De Vergelijking: De Barista

Stel je voor dat je koffie bestelt.

  • De Opslag (De Property): De koffiebonen en het water in de machine.
  • De Getter (De Methode): De handeling van koffie zetten. De barista pakt de bonen, maalt ze en geeft jou een kant-en-klaar product.

De Code: Een methode maken

We maken een nieuwe methode in onze class. Deze methode slaat zelf niks op, maar kijkt naar de andere data (Naam en Energie) en maakt daar een mooi tekstje van.

// BESTAND: DigitalPet.cs
class DigitalPet
{
    // De ruwe data (De Properties)
    public string Name { get; set; }
    public int Energy { get; set; }

    // De "Formatter" Methode
    // Dit is een actie: "Geef mij de mooie tekst"
    public string EnergyDisplay()
    {
        // We plakken tekst en data aan elkaar en geven het terug
        return "ðŸ”‹ " + this.Name + " - Level: " + this.Energy + "/100";
    }
}

Het gebruik in Program.cs

Omdat het een methode is, gebruiken we haakjes () om hem aan te roepen. Dit helpt je herinneren dat er code wordt uitgevoerd (het berekenen van de tekst).

// BESTAND: Program.cs
DigitalPet dino = new DigitalPet();
dino.Name = "Rex";
dino.Energy = 50;

// We roepen de methode aan met ()
Console.WriteLine(dino.EnergyDisplay());

// Output: ðŸ”‹ Rex - Level: 50/100

Voor de fijnproevers: Computed Properties

In C# is er nog een speciale truc. Je kunt een methode vermommen als een variabele. Dit heet een Computed Property. Je laat de haakjes () weg en gebruikt het woordje get.

// Ziet eruit als een variabele, maar voert stiekem code uit
public string Info
{
    get { return "Naam: " + Name; }
}
// Gebruik: Console.WriteLine(dino.Info); // Geen haakjes!

Dit mag je gebruiken, maar om verwarring te voorkomen raden we aan om voorlopig gewoon methodes met () te gebruiken voor acties.

Samenvatting

  • Gebruik een Getter Methode om data mooi te maken (formatteren) voordat je het aan de gebruiker toont.
  • Omdat het een methode is, gebruik je haakjes ().
  • Je kunt de Fat Arrow => gebruiken om de methode korter op te schrijven.
