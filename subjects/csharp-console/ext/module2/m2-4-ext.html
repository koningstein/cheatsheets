NuGet: De App Store voor Code

Tot nu toe heb je alle code zelf geschreven. Maar stel je voor dat je iets ingewikkelds moet doen, zoals een PDF maken, ingewikkelde wiskunde, of... vette graffiti letters in je console zetten. Moet je dat allemaal zelf programmeren?

Gelukkig niet. Miljoenen programmeurs hebben dit soort problemen al opgelost en delen hun code gratis. In de .NET wereld (C#) heet het systeem om deze code te delen NuGet.

De Vergelijking: LEGO

Als je een kasteel van LEGO bouwt, ga je niet zelf het plastic smelten om blokjes te maken. Je koopt kant-en-klare doosjes met blokjes.

  • Jouw project is het kasteel (jouw eigen creatie).
  • Een NuGet Package is een doosje met speciale blokjes (bijv. ramen of wielen) die iemand anders heeft gemaakt.

Waarom NuGet?

Waarom zou je het wiel opnieuw uitvinden? Als je een package gebruikt:

  • Bespaar je heel veel tijd.
  • Zitten er minder fouten in (want duizenden mensen hebben het al getest).
  • Kun jij je focussen op wat jouw game uniek maakt.

Hoe installeer je een Package?

In Visual Studio zit een ingebouwde winkel. We gaan als voorbeeld Figgle gebruiken voor ASCII-Art. Let goed op, want er zijn meerdere versies!

  1. Ga naar de Solution Explorer (rechts).
  2. Klik met de rechtermuisknop op je Project (bijv. TamagotchiGame).
  3. Kies in het menu: Manage NuGet Packages...
  4. Er opent een nieuw tabblad. Klik bovenaan op Browse.
  5. Typ in de zoekbalk: Figgle.
  6. BELANGRIJK: Kies het pakket Figgle.Fonts (Dus NIET alleen 'Figgle').
  7. Klik rechts op Install.

Waarom Figgle.Fonts? Het pakket 'Figgle' is alleen de motor. Het pakket 'Figgle.Fonts' bevat de daadwerkelijke lettertypes (zoals Standard). Als je die installeert, krijg je de motor er gratis bij.

Het Package gebruiken

Nu het pakket in je project zit, kun je de code gebruiken. Vaak moet je wel even de Namespace importeren (met using), anders weet C# niet waar de nieuwe code staat.

Het Doel: Een vet startscherm

We willen onze game beginnen met een groot logo.

// BESTAND: Program.cs
using Tamagotchi.Models;

// 1. BELANGRIJK: Importeer de namespace van het pakket
// Hierdoor kunnen we bij de class 'FiggleFonts'
using Figgle.Fonts;

// ... (Hieronder je Main code) ...

// Gebruik de code uit het pakket.
// De class heet 'FiggleFonts'.
string banner = FiggleFonts.Standard.Render("Dino Game");

Console.WriteLine(banner);

DigitalPet dino = new DigitalPet("Dino");
Console.WriteLine("Welkom bij " + dino.Name);

Waarom is dit handig?

Zonder dit pakket had je regel voor regel met Console.WriteLine(" ____ _ ...") moeten uittypen. Dat kost uren en als je de tekst wilt veranderen, moet je opnieuw beginnen. Nu doet Figgle al het werk voor je.

Wat kan er fout gaan? (Veelgemaakte Fouten)

⚠️ FiggleFonts is rood of onbekend

1. Heb je echt Figgle.Fonts geïnstalleerd en niet alleen Figgle?
2. Ben je using Figgle.Fonts; bovenaan vergeten?
3. Staat er wel echt FiggleFonts (meervoud met een s)?

Samenvatting

  • NuGet is de plek waar je kant-en-klare code van anderen kunt downloaden.
  • Je installeert packages via Manage NuGet Packages.
  • Let goed op welk pakket je kiest (soms zijn er losse pakketten voor data, zoals Figgle.Fonts).
  • Gebruik packages om tijd te besparen.
