NuGet: De App Store voor Code

Tot nu toe heb je alle code zelf geschreven. Maar stel je voor dat je iets ingewikkelds moet doen, zoals een PDF maken, een QR-code genereren of communiceren met een database. Moet je dat allemaal zelf programmeren?

Gelukkig niet. Miljoenen programmeurs hebben dit soort problemen al opgelost en delen hun code gratis. In de .NET wereld (C#) heet het systeem om deze code te delen NuGet.

De Vergelijking: LEGO

Als je een kasteel van LEGO bouwt, ga je niet zelf het plastic smelten om blokjes te maken. Je koopt kant-en-klare doosjes met blokjes.

  • Jouw project is het kasteel (jouw eigen creatie).
  • Een NuGet Package is een doosje met speciale blokjes (bijv. ramen of wielen) die iemand anders heeft gemaakt.

Waarom NuGet?

Waarom zou je het wiel opnieuw uitvinden? Als je een package gebruikt:

  • Bespaar je heel veel tijd.
  • Zitten er minder fouten in (want duizenden mensen hebben het al getest).
  • Kun jij je focussen op wat jouw game uniek maakt.

Hoe installeer je een Package?

In Visual Studio zit een ingebouwde winkel. We gaan als voorbeeld het pakket Newtonsoft.Json installeren. Dit is een wereldberoemd pakket om objecten om te zetten naar tekst.

  1. Ga naar de Solution Explorer (rechts).
  2. Klik met de rechtermuisknop op je Project (bijv. DigitalPetGame).
  3. Kies in het menu: Manage NuGet Packages...
  4. Er opent een nieuw tabblad. Klik bovenaan op Browse.
  5. Typ in de zoekbalk: Newtonsoft.Json.
  6. Klik op het bovenste resultaat en druk rechts op Install.

Als er een popup komt over wijzigingen of licenties, klik je gewoon op OK of I Accept.

Het Package gebruiken

Nu het pakket in je project zit, kun je de code gebruiken. Vaak moet je wel even de Namespace importeren (met using), anders weet C# niet waar de nieuwe code staat.

Het Doel: Opslaan als tekst

Stel, we willen ons huisdier opslaan in een tekstbestand. Een object (met haakjes en geheugenadressen) kun je niet zomaar opslaan. We moeten het omzetten naar JSON tekst.

Stap 1: Het Model (DigitalPet.cs)

Hier verandert niks. Dit is gewoon jouw eigen code.

namespace MyTamagotchiGame.Models;

public class DigitalPet
{
    public string Name { get; set; }
    public int Energy { get; set; }
}

Stap 2: Het Gebruik (Program.cs)

Hier gebruiken we de magische code van iemand anders.

using System;
using MyTamagotchiGame.Models;

// 1. BELANGRIJK: Importeer de namespace van het pakket
using Newtonsoft.Json;

// Maak een huisdier
DigitalPet dino = new DigitalPet();
dino.Name = "Dino";
dino.Energy = 100;

// Gebruik de code uit het pakket (JsonConvert)
// SerializeObject betekent: "Maak van dit object een stuk tekst"
string jsonText = JsonConvert.SerializeObject(dino);

Console.WriteLine(jsonText);
// Output: {"Name":"Dino","Energy":100}

Waarom is dit handig?

Zonder dit pakket had je zelf een functie moeten schrijven die: "{ \"Name\": \"" + dino.Name + "\" ... " aan elkaar plakt. Dat is moeilijk en foutgevoelig. Nu doet JsonConvert al het zware werk voor je.

Samenwerken (Dependencies)

Als je jouw project naar een klasgenoot stuurt (via Teams of Git), stuur je meestal niet de gedownloade packages mee (die map is veel te groot). Je stuurt alleen jouw eigen code.

Dit is geen probleem! Visual Studio ziet in jouw projectbestand (.csproj) welke packages er nodig zijn.

Zodra jouw klasgenoot het project opent en op "Start" drukt, downloadt Visual Studio razendsnel alle ontbrekende pakketten. Dit heet Package Restore.

Wat kan er fout gaan? (Veelgemaakte Fouten)

⚠️ Rode strepen onder de code

Heb je het pakket geïnstalleerd, maar geeft JsonConvert toch een rode streep?

  • Ben je using Newtonsoft.Json; vergeten bovenaan je bestand?
  • Heb je het pakket misschien in de Solution geïnstalleerd in plaats van in het Project?

⚠️ Verkeerde versie

Soms zijn er meerdere versies van een pakket. Voor schoolopdrachten is de nieuwste stabiele versie (Stable) meestal prima. Vink "Include prerelease" uit als je zeker wilt zijn dat het werkt.

Samenvatting

  • NuGet is de plek waar je kant-en-klare code van anderen kunt downloaden.
  • Je installeert packages via Manage NuGet Packages (rechtermuisknop op Project).
  • Je moet vaak een using toevoegen om de code te kunnen gebruiken.
  • Gebruik packages om tijd te besparen bij moeilijke taken (zoals JSON, Database, PDF).
