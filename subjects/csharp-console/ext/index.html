C# Console Curriculum (Extended)

Welkom bij de Extended Editie. In deze versie duiken we dieper in de stof met uitgebreide voorbeelden, achtergrondinformatie en extra oefeningen.

Fase 1: Fundamentele OOP

Klik op een module om de uitgebreide lesstof te starten.

Module 1

Basis OOP

Uitgebreide introductie tot Classes, Properties (Auto & Full), Methods, Constructors, Object Initializers en het this keyword.

 Module 2

Project & Debugging

Namespaces, Solution vs Project, File Scoped Namespaces, NuGet Packages en geavanceerd Debuggen (Locals, Watch, Call Stack).

 Module 3

Type Safety

Encapsulatie, Access Modifiers (private/public), Validatie in Setters, Strong Typing, Nullable Types en XML Documentatie.

 Module 4

Collections & Data

Generics (List<T>), Dictionaries, LINQ Basics (Where/Select), String Interpolation, JSON Serialization en File I/O.

 Module 5

Inheritance

Base & Derived Classes, Constructor Chaining (base), Overriding (virtual/override), Protected access en Sealed Classes.

 Module 6

Abstractie

Abstract Classes & Methods, Interfaces, Polymorfisme en het implementeren van Multiple Interfaces.

 Module 7

Advanced Concepts

Static Members & Classes, Extension Methods, Exception Handling (try/catch/throw) en een introductie tot Events & Delegates.
