Module 4-7: Opslaan & Laden (JSON)

Onze game werkt, maar als we afsluiten zijn alle dieren weg. Dat gaan we oplossen door de lijst op te slaan in een tekstbestand (JSON).

Stap 1: Namespaces (Game.cs)

C# heeft standaard geen verstand van JSON of bestanden schrijven. We moeten bibliotheken importeren. Zet deze twee regels helemaal bovenin Game.cs (boven `namespace Tamagotchi;`). 

using System.IO;         // Nodig voor File.WriteAllText en File.ReadAllText
using System.Text.Json;  // Nodig voor JsonSerializer

Stap 2: Opslaan (Serialize)

We maken een methode die de lijst met dieren omzet naar tekst. Zet deze methode helemaal onderaan in de `Game` class (onder de Start-methode, maar wel binnen de accolades van de class). 

    private void SaveGame()
    {
        // We stellen in dat de JSON netjes ingesprongen moet zijn (leesbaar voor mensen)
        var options = new JsonSerializerOptions
        {
            WriteIndented = true
        };

        // 1. Serialize: Maak van de lijst (_pets) een lange string (json)
        string json = JsonSerializer.Serialize(_pets, options);

        // 2. Schrijf die string naar een bestand genaamd "dieren.json"
        File.WriteAllText("dieren.json", json);

        Console.WriteLine("Spel succesvol opgeslagen!");
    }

Stap 3: Laden (Deserialize)

We hebben ook een methode nodig die het bestand leest en er weer een lijst van maakt. Zet deze methode ook onderaan in Game.cs. 

    private void LoadGame()
    {
        // Check altijd eerst of het bestand wel bestaat, anders crasht de game!
        if (File.Exists("dieren.json"))
        {
            // 1. Lees de tekst uit het bestand
            string json = File.ReadAllText("dieren.json");

            // 2. Deserialize: Maak van de tekst weer een List
            // De '?? new List...' betekent: als het bestand leeg of fout is, maak dan een lege lijst aan.
            _pets = JsonSerializer.Deserialize<List<DigitalPet>>(json) ?? new List<DigitalPet>();

            Console.WriteLine($"Savegame geladen: {_pets.Count} dieren gevonden.");
        }
        else
        {
            Console.WriteLine("Geen opgeslagen spel gevonden. We beginnen opnieuw.");
        }
    }

Stap 4: Integreren in de Start

De methodes doen niets als we ze niet aanroepen. We moeten LoadGame aanroepen bij het opstarten, en SaveGame bij het afsluiten.

Ga naar je Start() methode en pas hem op deze twee plekken aan:

4.1 Laden aan het begin

Zet dit helemaal bovenaan in de Start methode:

    public void Start()
    {
        // 1. Eerst proberen we te laden
        LoadGame();

        // Even wachten zodat de gebruiker het resultaat ziet
        _menu.WaitForEnter();

        _menu.ShowTitle();
        bool appRunning = true;

        // ... hieronder begint de while loop ...

4.2 Opslaan aan het einde

Ga in de while-loop naar de switch van het Hoofdmenu. Zoek case "Q" op en pas deze aan:

                case "Q": // Stoppen
                    // 2. Voordat we stoppen, slaan we op
                    SaveGame();

                    appRunning = false;
