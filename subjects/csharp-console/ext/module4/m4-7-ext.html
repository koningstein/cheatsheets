Module 4-7: Opslaan & Laden (JSON)

Onze game is leuk, maar als we afsluiten zijn de dieren dood. Dat kan niet! We gaan de dieren opslaan in een tekstbestand met JSON.

Stap 1: Namespaces (Game.cs)

Om bestanden te lezen en JSON te gebruiken, hebben we tools nodig. Zet dit helemaal bovenin Game.cs: 

using System.IO;         // Voor bestanden
using System.Text.Json;  // Voor JSON

Stap 2: Opslaan (Serialize)

We maken een methode SaveGame() in Game.cs. Deze pakt de lijst _pets en maakt er tekst van. 

private void SaveGame()
{
    // Opties voor mooie leesbare tekst
    var options = new JsonSerializerOptions { WriteIndented = true };

    // 1. Van Lijst naar Tekst (JSON)
    string json = JsonSerializer.Serialize(_pets, options);

    // 2. Schrijf naar bestand
    File.WriteAllText("dieren.json", json);

    Console.WriteLine("Opgeslagen!");
}

Stap 3: Laden (Deserialize)

We maken een methode LoadGame(). Deze doet het omgekeerde: van Tekst naar Lijst. We moeten wel checken of het bestand bestaat, anders crasht de app. 

private void LoadGame()
{
    if (File.Exists("dieren.json"))
    {
        // 1. Lees tekst
        string json = File.ReadAllText("dieren.json");

        // 2. Van Tekst naar Lijst
        _pets = JsonSerializer.Deserialize<List<DigitalPet>>(json);

        Console.WriteLine("Dieren ingeladen!");
    }
}

Stap 4: Integreren

Nu moeten we deze methodes nog aanroepen in de Start() methode. 

public void Start()
{
    // LADEN bij opstarten
    LoadGame();

    while (appRunning)
    {
        // ... (De loop) ...

        if (choice == "Q")
        {
            // OPSLAAN bij afsluiten
            SaveGame();
            appRunning = false;
        }
    }
}
