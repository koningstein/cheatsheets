Module 4-2: Input & Parsing

We gaan de applicatie interactief maken. We beginnen met een lege lijst. De gebruiker moet zelf via een menu dieren toevoegen en daarbij de statistieken (Energie/Honger) bepalen.

Stap 1: De Lege Lijst

We beginnen in Program.cs. We maken alleen het "mandje" (de List) aan. Deze is bij het opstarten leeg. 

using Figgle.Fonts;
using Tamagotchi.Models;

Console.WriteLine(FiggleFonts.Standard.Render("Dino Shelter"));

// We maken een LEGE lijst aan.
List<DigitalPet> pets = new List<DigitalPet>();

Stap 2: De Menu Loop

We bouwen een `while`-loop die blijft draaien. In die loop tonen we het menu en vragen we de gebruiker om een keuze. 

bool appRunning = true;

while (appRunning)
{
    // Toon de status (bij start is count 0)
    Console.WriteLine($"\n--- HOOFDMENU ({pets.Count} dieren) ---");
    Console.WriteLine("1. Nieuw dier toevoegen");
    Console.WriteLine("2. Alle dieren tonen");
    Console.WriteLine("Q. Stoppen");
    Console.Write("Maak uw keuze: ");

    string choice = Console.ReadLine().ToUpper();

    // Hieronder komt de switch...
}

Stap 3: Input & Parsing (Case 1)

Dit is de kern van de les. Als de gebruiker "1" kiest, vragen we om gegevens. Omdat `Console.ReadLine()` altijd tekst teruggeeft, moeten we de Energie en Honger omzetten naar getallen met `int.Parse()`.

Let op: De methode `int.Parse()` werkt alleen als de gebruiker cijfers typt. Als je letters typt, crasht de app (dit lossen we in latere modules op).

Plaats deze switch binnen de while-loop:

    switch (choice)
    {
        case "1":
            Console.Write("Naam: ");
            string name = Console.ReadLine();

            Console.Write("Start Energie (0-100): ");
            string energyText = Console.ReadLine();
            int energy = int.Parse(energyText); // <-- Van tekst naar getal

            Console.Write("Start Honger (0-100): ");
            string hungerText = Console.ReadLine();
            int hunger = int.Parse(hungerText); // <-- Van tekst naar getal

            // We maken het dier met de ingevoerde waardes
            DigitalPet newPet = new DigitalPet(name, energy, hunger);

            // En voegen hem toe aan de lijst
            pets.Add(newPet);

            Console.WriteLine($"{name} is toegevoegd!");
            break;

Stap 4: Lijst Tonen & Afsluiten

We voegen de optie toe om de lijst te bekijken. Als de lijst leeg is, zal de `foreach` loop simpelweg niets doen (wat logisch is). Ook voegen we de stop-optie toe. 

        case "2":
            Console.WriteLine("\n--- OVERZICHT ---");

            // Loop door alle dieren die de gebruiker heeft toegevoegd
            foreach (DigitalPet pet in pets)
            {
                Console.WriteLine(pet.EnergyDisplay());
            }

            Console.WriteLine("Druk op ENTER om terug te gaan.");
            Console.ReadLine();
            break;

        case "Q":
            appRunning = false; // Stop de loop
            Console.WriteLine("Tot ziens!");
            break;

        default:
            Console.WriteLine("Ongeldige keuze.");
            break;
    } // Einde switch
} // Einde while

Samenvatting

Je hebt nu een applicatie die begint met niets. De gebruiker bouwt de data zelf op. De belangrijkste techniek hier is Parsing: het omzetten van `string` (input) naar `int` (data).
