Module 4-6: Dictionaries & Eten Geven

We kunnen inloggen op een dier! Nu willen we het voeren. We maken een voorraadkast (Dictionary) in de Game en leren het dier eten.

Stap 1: De Voorraadkast (Game.cs)

In Game.cs maken we een Dictionary aan. Dit is onze pantry. We zetten dit bovenaan bij de andere private fields. 

// In Game.cs

private Dictionary<string, Food> _pantry = new Dictionary<string, Food>();

// Tip: In de Constructor (public Game()) kun je hem vullen:
public Game()
{
    _pantry.Add("Appel", new Food("Appel", 10));
    _pantry.Add("Brood", new Food("Brood", 20));
}

Stap 2: Logica in de Class (DigitalPet.cs)

De Game-class mag niet zelf aan de Energy variabele van het dier zitten (dat is private!). We moeten het dier netjes vragen om te eten.

Open Models/DigitalPet.cs en voeg toe:

// In DigitalPet.cs

public string Feed(Food food)
{
    this.Energy += food.EnergyGain;
    this.Hunger -= 10;

    if (this.Hunger < 0) this.Hunger = 0;

    return $"{this.Name} at een {food.Name}. (+{food.EnergyGain} Energie)";
}

Stap 3: Het Menu Uitbreiden (Game.cs)

Nu voegen we de optie "Voeren" toe aan het Actie Menu (de `else` in je loop). We tonen de keys (namen) en laten de gebruiker kiezen. 

        // In het else blok van Game.cs, case "2":
