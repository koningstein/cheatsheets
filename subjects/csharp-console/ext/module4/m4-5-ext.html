Module 4-5: State Management

We gaan zorgen dat de knoppen werken. We introduceren het concept "Inloggen" op een dier.

Stap 1: Menu Uitbreiden (Menu.cs)

We moeten eerst onze Ober (Menu) wat nieuwe trucjes leren.

1.1 Getallen vragen

Voor Energie en Honger hebben we getallen nodig. Voeg deze hulp-methode toe:

    // In Menu.cs
    public int AskInt(string question)
    {
        Console.Write(question + " ");
        // Probeer tekst om te zetten naar getal, anders 50
        if (int.TryParse(Console.ReadLine(), out int result))
            return result;
        return 50;
    }

1.2 Het Actie Menu

Als we ingelogd zijn op een dier, zien we een ander menu. Hier gebruiken we 0 om terug te gaan.

    // In Menu.cs
    public void ShowPetMenu(string name, int energy, int hunger)
    {
        Console.WriteLine($"\n--- MENU: {name} ---");
        Console.WriteLine($"Energie: {energy} | Honger: {hunger}");
        Console.WriteLine("0. Terug (Uitloggen)");
    }

// In Game.cs (bovenin bij de fields)

Stap 3: De Logica Splitsen

We gaan de Start() methode verbouwen. We kijken steeds: is `_currentPet` leeg (null)?

Zorg dat je `Start()` er zo uit komt te zien (de inhoud van de switches doen we hierna):

    public void Start()
    {
        _menu.ShowTitle();
        bool appRunning = true;

        while (appRunning)
        {
            // SITUATIE A: Geen dier (Hoofdmenu)
            if (_currentPet == null)
            {
                _menu.ShowMainMenu(_pets.Count);
                string choice = _menu.AskString("Keuze:");

                // ... Hier komt Switch A (zie Stap 4)
            }
            // SITUATIE B: Wel dier (Actie Menu)
            else
            {
                _menu.ShowPetMenu(_currentPet.Name, _currentPet.Energy, _currentPet.Hunger);
                string action = _menu.AskString("Actie:");

                // ... Hier komt Switch B (zie Stap 5)
            }
        }
    }

Stap 4: Het Hoofdmenu Invullen

Nu vullen we de eerste switch in. Hier regelen we toevoegen, de lijst tonen en inloggen.

            // Switch A (In het if-blok)
            switch (choice.ToUpper())
            {
                case "1": // Toevoegen
                    string name = _menu.AskString("Naam:");
                    int energy = _menu.AskInt("Energie (0-100):");
                    int hunger = _menu.AskInt("Honger (0-100):");
                    _pets.Add(new DigitalPet(name, energy, hunger));
                    Console.WriteLine("Toegevoegd!");
                    _menu.WaitForEnter();
                    break;

                case "2": // Lijst Tonen
                    Console.WriteLine("\n--- ALLE DIEREN ---");
                    foreach (var pet in _pets)
                    {
                        Console.WriteLine(pet.EnergyDisplay());
                    }
                    _menu.WaitForEnter();
                    break;

                case "3": // Zoeken & Inloggen
                    string search = _menu.AskString("Wie zoek je?");
                    _currentPet = _pets.FirstOrDefault(p => p.Name.ToLower() == search.ToLower());

                    if (_currentPet == null)
                    {
                        Console.WriteLine("Niet gevonden.");
                        _menu.WaitForEnter();
                    }
                    break;

                case "Q": // Stoppen
                    appRunning = false;
                    break;

                default:
                    Console.WriteLine("Ongeldige keuze.");
                    _menu.WaitForEnter();
                    break;
            }

Stap 5: Het Actie Menu Invullen

Als we ingelogd zijn, willen we ook weer terug kunnen. Dat doen we door `_currentPet` weer op `null` te zetten.

            // Switch B (In het else-blok)
            switch (action)
            {
                case "0": // TERUG
                    _currentPet = null; // Door dit op null te zetten, gaan we terug naar het hoofdmenu
                    break;

                default:
                    Console.WriteLine("Ongeldige keuze.");
                    _menu.WaitForEnter();
                    break;
            }
