Les 3.3: Constants & Readonly

We vervangen "Magic Numbers" (losse getallen) door constanten en gebruiken `readonly` voor data die na aanmaak niet meer mag wijzigen.

Voeg const variabelen toe bovenin DigitalPet.cs. Deze waardes staan vast voor de hele applicatie.

public class DigitalPet
{
    public const int MaxEnergy = 100;
    public const int MinEnergy = 0;
    public const int SleepEnergyGain = 20;

    // ... fields en properties

Stap 2: Implementeren en String Interpolation

Vervang de harde getallen in je code door de constanten. Gebruik de $ syntax voor tekst.

    // Voorbeeld in Sleep() methode:
    public string Sleep()
    {
        this.IsSleeping = true;
        this.Energy += SleepEnergyGain; // Gebruik constant
        UpdateStatus();

        // String Interpolation ($)
        return $"Zzz... (+{SleepEnergyGain} Energie)";
    }

Vergeet niet ook je Property validatie aan te passen naar `MinEnergy` en `MaxEnergy`.

Stap 3: Readonly Property

Gebruik een Read-only Property (alleen `get`) voor waardes die uniek zijn per object, maar na de constructor vaststaan.

    // Geen 'set', ook geen private set.
    public DateTime BirthDate { get; }

Stap 4: Constructor Update

De constructor is de enige plek waar je een read-only property mag vullen.

    public DigitalPet(string name, int energy = 50, int hunger = 0, bool isSleeping = false)
    {
        this.Name = name;
        this.Energy = energy;
        this.Hunger = hunger;
        this.IsSleeping = isSleeping;

        this.BirthDate = DateTime.Now; // Hier mag het!
    }

Stap 5: Aanroepen in Program.cs

DigitalPet dino = new DigitalPet("Rex");

// Const: Via de Class naam
Console.WriteLine($"Max: {DigitalPet.MaxEnergy}");

// Readonly: Via het Object
Console.WriteLine($"Geboren: {dino.BirthDate}");
