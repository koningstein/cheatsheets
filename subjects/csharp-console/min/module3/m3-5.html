Les 3.5: Documentatie (XML Comments)

We voegen XML-documentatie toe zodat Visual Studio IntelliSense (pop-ups) kan tonen met uitleg over onze code.

XML Comments

Door `///` boven een class, methode of property te typen, kun je documentatie toevoegen.

  • <summary>: Korte beschrijving.
  • <remarks>: Extra details/waarschuwingen.
  • <param>: Uitleg van input parameters.
  • <returns>: Uitleg van het resultaat.
