Setter & Validatie

Data beschermen

Soms wil je controleren wat er in een variabele wordt gestopt. Bijvoorbeeld: energie mag nooit negatief zijn. Omdat properties nu nog openbaar zijn, gebruiken we een Methode om dit te controleren.

De Validatie Methode

We maken een methode in de class die kijkt of de waarde geldig is, voordat hij deze opslaat.

// BESTAND: DigitalPet.cs
public void SetEnergy(int amount)
{
    if (amount < 0)
    {
        Console.WriteLine("Fout! Mag niet negatief zijn.");
        this.Energy = 0; // Corrigeer naar 0
    }
    else
    {
        this.Energy = amount; // Het is veilig, sla op
    }
}

Het gebruik

In Program.cs gebruik je nu deze methode in plaats van de property rechtstreeks aan te passen.

// BESTAND: Program.cs
DigitalPet dino = new DigitalPet();

// Niet veilig:
// dino.Energy = -50;

// Veilig (via de methode):
dino.SetEnergy(-50);
// Output: Fout! Mag niet negatief zijn. De energie is nu 0.
