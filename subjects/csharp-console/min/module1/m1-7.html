Constructors

Een vliegende start

Tot nu toe maakten we een huisdier aan en vulden we daarna pas de naam in. Maar een naamloos huisdier zou niet moeten bestaan. We willen afdwingen dat een object verplicht data krijgt bij de geboorte.

Hiervoor gebruiken we de Constructor. Dit is een methode die automatisch wordt uitgevoerd zodra je new gebruikt.

Syntax Regels

  1. De methode heet exact hetzelfde als de Class.
  2. Er is geen return type (ook geen void).
// BESTAND: DigitalPet.cs
class DigitalPet
{
    public string Name { get; set; }
    public int Energy { get; set; }

    // De Constructor
    public DigitalPet(string name)
    {
        this.Name = name;    // Verplichte data opslaan
        this.Energy = 50;    // Standaard startwaarde instellen
    }
}

Het Gebruik

Omdat we een constructor hebben gemaakt die een parameter verwacht, mag je geen lege haakjes meer gebruiken.

// BESTAND: Program.cs

// FOUT: De class eist nu een naam.
// DigitalPet pet = new DigitalPet();

// GOED: We geven de naam mee bij de start.
DigitalPet dino = new DigitalPet("Rex");

Console.WriteLine(dino.Name);   // Output: Rex
Console.WriteLine(dino.Energy); // Output: 50 (Automatisch ingesteld)

Tip: 'ctor' Snippet

In Visual Studio hoef je dit niet helemaal zelf te typen. Typ simpelweg ctor en druk twee keer op de TAB-toets. Visual Studio genereert dan automatisch de constructor code.
