Constructors

Een object geboren laten worden

Tot nu toe maakten we een huisdier aan met new DigitalPet() en vulden we daarna pas de naam in. Maar bestaat er zoiets als een naamloos huisdier? Eigenlijk willen we afdwingen dat zodra een huisdier wordt gemaakt, hij verplicht een naam krijgt.

Hiervoor gebruiken we de Constructor. Dit is een methode die automatisch wordt uitgevoerd op het moment dat je het keyword new gebruikt.

Syntax

Een constructor heeft twee unieke eigenschappen:

  1. Hij heeft exact dezelfde naam als de class.
  2. Hij heeft geen return type (ook geen void).
class DigitalPet
{
    public string Name { get; set; }
    public int Energy { get; set; }

    // De Constructor
    public DigitalPet(string name)
    {
        this.Name = name;
        this.Energy = 50; // We geven direct een startwaarde mee
    }
}

Verplichte data

Omdat we nu een constructor hebben gemaakt die een parameter verwacht (`string name`), vervalt de standaard 'lege' constructor. Je code wordt nu strenger en veiliger:

// FOUT: Dit mag niet meer! De class eist nu een naam.
// DigitalPet pet = new DigitalPet();

// GOED: We geven de verplichte naam mee bij de geboorte.
DigitalPet myPet = new DigitalPet("Blafmans");

Tip: 'ctor' Snippet

In Visual Studio hoef je de constructor niet helemaal zelf te typen. Typ simpelweg ctor en druk twee keer op de TAB-toets. Visual Studio genereert dan automatisch de constructor code voor je class.
