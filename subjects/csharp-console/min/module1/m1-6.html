Getters: Data Formatteren

Mooie data tonen

Soms wil je data niet "rauw" tonen (bijv. 50), maar mooi opgemaakt (bijv. "Level: 50/100"). Hiervoor maken we een speciale Getter Methode in de class.

De Code

Je maakt een methode die niets opslaat, maar data pakt, samenvoegt en teruggeeft (return).

// BESTAND: DigitalPet.cs
public string EnergyDisplay()
{
    // Plak tekst en variabelen aan elkaar
