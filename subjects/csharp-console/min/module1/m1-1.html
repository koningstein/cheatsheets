Class

Wat is een Class?

In C# is alles gebaseerd op objecten. Een class is de blauwdruk (het ontwerp) van zo'n object. Het beschrijft welke gegevens (data) en functionaliteiten (gedrag) een object heeft, maar het is nog niet het object zelf.

Voorbeeld: Digital Pet

We gaan een virtueel huisdier maken (een soort Tamagotchi). We maken hiervoor een class DigitalPet. Deze class beschrijft dat elk huisdier een naam en een energieniveau heeft. Maar de class zelf leeft niet! Pas als we de class gebruiken om een Object te maken (bijvoorbeeld "Dino" of "Fluffy"), hebben we een echt digitaal huisdier.

Hoe ziet een Class eruit?

Een class in C# definieer je met het keyword class, gevolgd door de naam. De inhoud van de class staat tussen accolades { }.

class DigitalPet
{
    // Hier komt de inhoud (eigenschappen en gedrag)
}

Belangrijke regels voor een Class

1. Naamgeving

In C# gebruiken we PascalCase voor class-namen. Dit betekent dat elk woord met een hoofdletter begint.

  • Goed: DigitalPet, GameManager, Program
  • Fout: digitalPet, digital_pet, Digitalpet

2. Bestandsnaam

In C# is het de conventie (standaardregel) dat de bestandsnaam exact overeenkomt met de class-naam, gevolgd door de extensie .cs.

Als je class DigitalPet heet, noem je het bestand DigitalPet.cs.


3. Structuur & Scope

Net als in PHP gebruiken we accolades { } om het blok code aan te geven. In C# is het gebruikelijk (en in latere modules verplicht) om classes netjes te groeperen, maar voor nu volstaat de basis definitie.

class DigitalPet
{
    // Eigenschappen (bijv. Name, Energy) en methodes (bijv. Feed) komen hier
}
class FoodItem
{
    // Een class voor het voedsel van je huisdier
}
