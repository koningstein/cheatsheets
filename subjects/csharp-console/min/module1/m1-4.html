Methoden

Actie ondernemen

Properties zijn de zelfstandige naamwoorden (Naam, Energie), maar Methoden zijn de werkwoorden. Methods zorgen ervoor dat een object iets kan doen (bijv. zichzelf voorstellen).

Een method maken

Je maakt een method in je Class bestand. Het lijkt op een normale functie, maar je hebt toegang tot de data van het object.

// BESTAND: DigitalPet.cs
class DigitalPet
{
    public string Name { get; set; }
    public int Energy { get; set; }

    // Een methode die tekst teruggeeft (string)
    public string GetDescription()
    {
        // 'this' verwijst naar de properties van DIT object
        return "Ik ben " + this.Name + " en ik heb " + this.Energy + " energie.";
    }
}

Het 'this' keyword

Met het woordje this zeg je eigenlijk: "Mijn".
this.Name betekent: "Mijn eigen naam". Zo weet het object welke data hij moet pakken.

Methoden aanroepen

In Program.cs gebruik je de punt . om de methode te starten. Vergeet de haakjes () niet!

// BESTAND: Program.cs

// 1. Maak Dino
DigitalPet dino = new DigitalPet();
dino.Name = "Rex";
dino.Energy = 100;

// 2. Maak Fluffy
DigitalPet fluffy = new DigitalPet();
fluffy.Name = "Fluffy";
fluffy.Energy = 80;

// 3. Roep de methoden aan
// Dino gebruikt ZIJN data, Fluffy gebruikt HAAR data.
Console.WriteLine(dino.GetDescription());
Console.WriteLine(fluffy.GetDescription());

Console.ReadLine();

Return vs Void

Let op het return-type:

  • string / int: De methode geeft een antwoord terug (zoals hierboven). Je moet dit opvangen of printen.
  • void: De methode doet alleen iets (actie) en geeft niks terug.
