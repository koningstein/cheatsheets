Methoden

Actie ondernemen

Properties zijn de zelfstandige naamwoorden (de Naam, de Energie), maar Methoden (Methods) zijn de werkwoorden. Methods zorgen ervoor dat een object iets kan doen.

Een method maken

Een method in een class lijkt erg op een normale functie. Het heeft een zichtbaarheid (public), een return type en een naam.

class DigitalPet
{
    public string Name { get; set; }

    // Een simpele methode zonder return waarde (void)
    public void Bark()
    {
        Console.WriteLine("Woef! Woef!");
    }
}

Methoden aanroepen

Net als bij properties gebruiken we de punt . om een methode te gebruiken. Vergeet de haakjes () niet!

DigitalPet dog = new DigitalPet();
dog.Name = "Blafmans";

// Roep de methode aan
dog.Bark();

// Output: Woef! Woef!

Parameters

Je kunt ook informatie meegeven aan een method. Bijvoorbeeld, hoeveel keer moet de hond blaffen?

public void Walk(int kilometers)
{
    Console.WriteLine("We hebben " + kilometers + " kilometer gewandeld.");
}
dog.Walk(5); // Output: We hebben 5 kilometer gewandeld.

Het 'this' keyword

Vaak wil je in een method de eigen properties van het object gebruiken. Bijvoorbeeld: de hond moet zijn eigen naam zeggen. Hiervoor gebruiken we this.

public void Introduce()
{
    // 'this.Name' verwijst naar de property Name van DIT specifieke object
    Console.WriteLine("Hallo, ik ben " + this.Name);
}
