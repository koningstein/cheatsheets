Een object zonder data is leeg. We willen dat onze DigitalPet eigenschappen heeft, zoals een naam en een energieniveau. In C# gebruiken we hiervoor Properties.

Je definieert deze in je Class bestand (DigitalPet.cs).

Auto-Properties

De snelste manier is de "Auto-Implemented Property". Je voegt { get; set; } toe achter de naam.

// BESTAND: DigitalPet.cs
class DigitalPet
{
    // Een property voor tekst (de naam)
    public string Name { get; set; }

    // Een property voor een geheel getal (energie)
    public int Energy { get; set; }
}

Get en Set

  • get: De waarde mag uitgelezen worden (Read).
  • set: De waarde mag overschreven worden (Write).

Waardes toewijzen en uitlezen

In Program.cs gebruik je de punt-notatie . om bij de properties te komen. Je kunt meerdere objecten maken die elk hun eigen waardes hebben.

// BESTAND: Program.cs

// 1. Maak Dino en geef hem eigenschappen (SET)
DigitalPet dino = new DigitalPet();
dino.Name = "Rex";
dino.Energy = 100;

// 2. Maak Fluffy en geef haar eigenschappen (SET)
DigitalPet fluffy = new DigitalPet();
fluffy.Name = "Fluffy";
fluffy.Energy = 80;

// 3. Lees de data uit en toon het op het scherm (GET)
Console.WriteLine("Dier 1 heet: " + dino.Name);
Console.WriteLine("Dier 2 heet: " + fluffy.Name);

Console.ReadLine(); // Wacht op enter

Onafhankelijkheid

Belangrijk: Omdat we twee keer new hebben gebruikt, zijn dino en fluffy compleet gescheiden. Als je de energie van Dino aanpast, blijft de energie van Fluffy gelijk.
