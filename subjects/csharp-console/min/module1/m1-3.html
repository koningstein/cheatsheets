Een object zonder data is leeg. We willen dat onze DigitalPet eigenschappen heeft, zoals een naam en een energieniveau. In C# gebruiken we hiervoor Properties.

Een Property is publiek toegankelijk, heeft een datatype (zoals string of int) en een naam.

Auto-Properties

De snelste manier om een property te maken is de "Auto-Implemented Property". Je hoeft niet na te denken over hoe de data wordt opgeslagen, C# regelt dit voor je achter de schermen.

class DigitalPet
{
    // Een property voor tekst (de naam)
    public string Name { get; set; }

    // Een property voor een geheel getal (energie)
    public int Energy { get; set; }
}

Get en Set

Je ziet achter de property { get; set; } staan. Dit zijn de toegangsrechten:

  • get: De waarde mag uitgelezen worden (Read).
  • set: De waarde mag overschreven worden (Write).

Waardes toewijzen (Set)

Nadat je een object hebt aangemaakt met new, kun je de properties vullen. Je gebruikt de punt-notatie om bij de property te komen.

DigitalPet myPet = new DigitalPet();

// We schrijven data naar de properties (SET)
myPet.Name = "Dino";
myPet.Energy = 100;

Waardes uitlezen (Get)

Wil je de naam van je huisdier op het scherm tonen? Dan "haal" je de waarde op.

// We lezen de data (GET)
Console.WriteLine(myPet.Name);

// Output: Dino
