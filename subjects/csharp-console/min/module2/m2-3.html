Mappenstructuur

Orde houden

Als je veel classes hebt, stop je die niet allemaal in de hoofdmap. Net als op je computer gebruik je Mappen. Een standaardnaam voor een map met data-objecten (zoals je Pet en Food) is Models.

1. Map aanmaken

Klik rechts op je Project -> Add -> New Folder. Noem deze Models. Sleep je classes hierin (of maak nieuwe aan).

2. Namespace aanpassen

De namespace moet altijd de mapstructuur volgen. De namespace was eerst Tamagotchi, maar omdat het bestand nu in een map zit, plak je er .Models achter.

// BESTAND: Models/DigitalPet.cs
namespace Tamagotchi.Models; // <-- Was: namespace Tamagotchi;

public class DigitalPet
{
    // ...
}

3. Using toevoegen

In Program.cs (die buiten de map staat) moet je nu aangeven dat hij in de map moet kijken.

// BESTAND: Program.cs
using Tamagotchi.Models; // <-- Één keer importeren is genoeg

// Nu kun je ALLE classes uit die map gebruiken
DigitalPet dino = new DigitalPet("Rex");
Food appel = new Food("Appel", 10);

Console.WriteLine("We hebben " + dino.Name + " en een " + appel.Name);
