NuGet Packages

Het wiel niet opnieuw uitvinden

Als programmeur hoef je niet alles zelf te schrijven. Wil je complexe berekeningen doen, PDF's genereren of JSON bestanden inlezen? Grote kans dat iemand anders dat al heeft geprogrammeerd en gedeeld.

In de .NET wereld heet de pakketbeheerder NuGet (vergelijkbaar met Composer voor PHP of NPM voor JavaScript).

NuGet Package Manager

In Visual Studio kun je heel makkelijk pakketten installeren zonder commando's te typen:

  1. Rechtermuisknop op je Project (niet de Solution).
  2. Kies Manage NuGet Packages...
  3. Ga naar het tabblad Browse en zoek je library.

Voorbeeld: Newtonsoft.Json

Een van de populairste packages is Newtonsoft.Json. Hiermee kun je objecten heel makkelijk omzetten naar tekst (Serialize) en andersom.

Code gebruiken

Na installatie kun je de code direct gebruiken. Vaak moet je wel even de juiste namespace importeren.

using Newtonsoft.Json; // De namespace van het pakket

DigitalPet myPet = new DigitalPet("Dino");

// Het object omzetten naar een JSON string
string json = JsonConvert.SerializeObject(myPet);

Console.WriteLine(json);
// Output: {"Name":"Dino","Energy":50}

Dependencies

NuGet houdt in je projectbestand (.csproj) bij welke versies je gebruikt. Als je je project naar een teamgenoot stuurt, downloadt Visual Studio automatisch de ontbrekende pakketten (Restore).
