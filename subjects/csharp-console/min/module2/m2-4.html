NuGet Packages

Code lenen van anderen

Soms heb je functionaliteit nodig die te complex is om zelf te bouwen (zoals ASCII Art genereren). Hiervoor gebruiken we NuGet Packages.

1. Installeren (Let op!)

Er zijn meerdere pakketten. Wij willen de fonts hebben.

  1. Rechtsklik op je Project -> Manage NuGet Packages.
  2. Zoek naar Figgle.
  3. BELANGRIJK: Kies het pakket Figgle.Fonts (Dus NIET alleen 'Figgle').
  4. Klik op Install.

2. Gebruiken (De Using)

Om de code te gebruiken, moet je de specifieke namespace van het pakket importeren.

// BESTAND: Program.cs
using Figgle.Fonts; // <-- Importeer de namespace

// Nu herkent hij de class 'FiggleFonts'
string banner = FiggleFonts.Standard.Render("Dino Game");

Console.WriteLine(banner);
