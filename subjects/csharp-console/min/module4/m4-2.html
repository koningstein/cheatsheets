Module 4-2: Input & Parsing

Alle invoer via Console.ReadLine() is standaard tekst (string). Om met getallen te werken (zoals Energie), moeten we de tekst omzetten. Dit heet Parsing.

1. Tekst Inlezen

Gebruik Console.ReadLine() om te wachten tot de gebruiker iets typt en op Enter drukt.

Console.Write("Naam: ");
string? input = Console.ReadLine();

// Check op lege invoer
if (string.IsNullOrWhiteSpace(input))
{
    // ... stop of probeer opnieuw ...
}

2. Parsing (String naar Int)

Je kunt tekst niet zomaar in een int variabele stoppen. Gebruik int.TryParse. Dit is veiliger dan `int.Parse` omdat het niet crasht als de gebruiker "hallo" typt in plaats van een cijfer. 

Console.Write("Voer energie in (0-100): ");
string numberInput = Console.ReadLine() ?? "0";

// TryParse probeert de tekst om te zetten.
// Het resultaat komt in de variabele 'energy' via de 'out' parameter.
int energy;
if (int.TryParse(numberInput, out energy))
{
    Console.WriteLine($"Gekozen energie: {energy}");
}
else
{
    Console.WriteLine("Dat was geen geldig getal. We gebruiken 0.");
    energy = 0;
}
