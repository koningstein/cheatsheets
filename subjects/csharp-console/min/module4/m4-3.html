Module 4-3: Zoeken met LINQ

Om specifieke items in een lijst te vinden, gebruiken we geen foreach loop met een if. We gebruiken LINQ (Language Integrated Query). Dit is korter en leesbaarder.

1. Namespace toevoegen

LINQ functies werken alleen als je deze regel helemaal bovenaan je bestand zet:

using System.Linq;

2. FirstOrDefault

De meest gebruikte zoekfunctie is FirstOrDefault. Hij zoekt het eerste item dat aan de voorwaarde voldoet. Vindt hij niets? Dan geeft hij null terug. 

Console.Write("Wie zoek je? ");
string search = Console.ReadLine() ?? "";

// Lambda Expressie: p => p.Name == search
// Lees als: "Zoek het dier 'p' waarvan de naam gelijk is aan 'search'"
DigitalPet? foundPet = pets.FirstOrDefault(p => p.Name == search);

3. Case Insensitive Zoeken

Strings zijn hoofdlettergevoelig ("rex" is niet "Rex"). Om dit op te lossen, zetten we alles om naar kleine letters (`ToLower`).

DigitalPet? foundPet = pets.FirstOrDefault(p => p.Name.ToLower() == search.ToLower());

4. Null Check (Veiligheid)

Als het dier niet bestaat, is de variabele null. Je moet dit controleren voordat je het gebruikt, anders crasht de app.

if (foundPet != null)
{
    // Gevonden!
    Console.WriteLine($"Gevonden: {foundPet.Name} heeft {foundPet.Energy} energie.");
}
else
{
    // Niet gevonden (foundPet is null)
    Console.WriteLine("Dier niet gevonden.");
}
