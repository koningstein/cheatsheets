Les 4.1: Lists & Generics

We vervangen losse variabelen en arrays door een List. Hiermee kunnen we dynamisch objecten toevoegen zonder vooraf de hoeveelheid te weten.

Stap 1: De Lijst Declareren

In plaats van een Array (`[]`) gebruiken we `List<T>`. Tussen de haakjes geef je het type aan.

using Tamagotchi.Models;

// OUDE MANIER (Array - Vaste grootte, crasht bij te veel items)
// DigitalPet[] pets = new DigitalPet[5];

// NIEUWE MANIER (List - Groeit automatisch mee)
List<DigitalPet> pets = new List<DigitalPet>();

Stap 2: De Loop (Toevoegen & Tonen)

We maken een interactieve loop in Program.cs. Hierin combineren we drie acties:

  1. Input: Gebruik Console.ReadLine().
  2. Process: Gebruik .Add() om een nieuw object in de lijst te stoppen.
  3. Output: Gebruik foreach om de lijst te tonen.
Console.WriteLine("Voer namen in. Druk op ENTER om te stoppen.");

while (true)
{
    Console.Write("\nNaam: ");
    string? input = Console.ReadLine();

    // Stop conditie
    if (string.IsNullOrWhiteSpace(input)) break;

    // 1. Toevoegen aan lijst
    pets.Add(new DigitalPet(input));

    // 2. Lijst weergeven
    Console.WriteLine($"\n--- Aantal dieren: {pets.Count} ---");

    foreach (DigitalPet pet in pets)
    {
        Console.WriteLine(pet.EnergyDisplay());
    }
}

Samenvatting Code

Dit is hoe je Program.cs er nu uit moet zien:

using Figgle.Fonts;
using Tamagotchi.Models;

Console.WriteLine(FiggleFonts.Standard.Render("Dino Shelter"));

List<DigitalPet> pets = new List<DigitalPet>();

while (true)
{
    Console.Write("Naam nieuw dier (of ENTER om te stoppen): ");
    string? input = Console.ReadLine();

    if (string.IsNullOrWhiteSpace(input)) break;

    pets.Add(new DigitalPet(input));

    foreach (DigitalPet pet in pets)
    {
        Console.WriteLine($"- {pet.Name} (Energy: {pet.Energy})");
    }
}
