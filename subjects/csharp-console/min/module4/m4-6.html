Module 4-6: Dictionaries

We gebruiken een Dictionary<Key, Value> voor de voorraadkast. Dit maakt het opzoeken van items (op basis van naam) heel snel.

{

Stap 2: Voorraadkast & Constructor (Game.cs)

Maak de Dictionary aan en vul deze in de constructor.

// Field
private Dictionary<string, Food> _pantry = new Dictionary<string, Food>();

// Constructor (Voeg toe boven Start)
public Game()
{
    _pantry.Add("Appel", new Food("Appel", 10));
    _pantry.Add("Brood", new Food("Brood", 20));
    _pantry.Add("Biefstuk", new Food("Biefstuk", 50));
}

Stap 3: Feed Methode (DigitalPet.cs)

Het dier verwerkt zelf het eten.

public string Feed(Food food)
{
    this.Energy += food.EnergyGain;
    this.Hunger -= 10;
    if (this.Hunger < 0) this.Hunger = 0;

    return $"{this.Name} at een {food.Name}. (+{food.EnergyGain} Energie)";
}
