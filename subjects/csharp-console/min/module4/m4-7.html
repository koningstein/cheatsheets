Module 4-7: JSON Opslaan & Laden

We gebruiken System.Text.Json om de lijst met objecten op te slaan in een tekstbestand.

Stap 1: Namespaces

Voeg toe bovenin Game.cs:

using System.IO;
using System.Text.Json;

Stap 2: Opslaan (Serialize)

Maak objecten plat tot tekst (JSON) en schrijf naar schijf.

private void SaveGame()
{
    var options = new JsonSerializerOptions { WriteIndented = true };
    string json = JsonSerializer.Serialize(_pets, options);
    File.WriteAllText("dieren.json", json);
    Console.WriteLine("Opgeslagen!");
}

Stap 3: Laden (Deserialize)

Lees tekst van schijf en maak er weer objecten van. Let op de null-check (`??`).

private void LoadGame()
{
    if (File.Exists("dieren.json"))
    {
        string json = File.ReadAllText("dieren.json");
        // Als bestand leeg is, maak nieuwe lijst
        _pets = JsonSerializer.Deserialize<List<DigitalPet>>(json) ?? new List<DigitalPet>();
        Console.WriteLine("Geladen.");
    }
}

Stap 4: Integratie

Roep de functies aan in Start().

public void Start()
{
    LoadGame(); // 1. Laden bij start
    _menu.WaitForEnter();

    // ... code ...

    while (appRunning)
    {
         // ... code ...
         case "Q":
             SaveGame(); // 2. Opslaan bij stop
             appRunning = false;
             break;
    }
}
