Module 4-4: Architectuur (Classes)

Program.cs wordt onoverzichtelijk. We splitsen de code op in drie delen: Data (Models), Weergave (Menu) en Logica (Game).

Locatie: Maak Menu.cs en Game.cs aan in de hoofdmap (naast Program.cs).

Stap 1: UI Class (Menu.cs)

Deze class regelt alle `Console` input en output. 

using Figgle.Fonts;

namespace Tamagotchi;

public class Menu
{
    public void ShowTitle()
    {
        Console.Clear();
        try { Console.WriteLine(FiggleFonts.Standard.Render("Dino Shelter")); }
        catch { Console.WriteLine("=== DINO SHELTER ==="); }
    }

    public void ShowMainMenu(int petCount)
    {
        Console.WriteLine($"\n--- HOOFDMENU ({petCount} dieren) ---");
        Console.WriteLine("1. Nieuw dier");
        Console.WriteLine("2. Bekijk alle dieren");
        Console.WriteLine("3. Login (Selecteer dier)");
        Console.WriteLine("Q. Stoppen");
    }

    public string AskString(string question)
    {
        Console.Write(question + " ");
        return Console.ReadLine() ?? "";
    }

    public void WaitForEnter()
    {
        Console.WriteLine("\nDruk op ENTER om verder te gaan...");
        Console.ReadLine();
    }
}

Stap 2: Logic Class (Game.cs)

Deze class beheert de lijst en de loop. Hij gebruikt de `Menu` class om dingen te tonen. 

using Tamagotchi.Models;

namespace Tamagotchi;

public class Game
{
    private Menu _menu = new Menu();
    private List<DigitalPet> _pets = new List<DigitalPet>();

    public void Start()
    {
        _menu.ShowTitle();
        bool appRunning = true;

        while (appRunning)
        {
            // Roep de UI aan
            _menu.ShowMainMenu(_pets.Count);
            string choice = _menu.AskString("Keuze:");

            // Simpele logica voor nu (wordt uitgebreid in 4-5)
            if (choice.ToUpper() == "Q")
            {
                appRunning = false;
            }
        }
    }
}

Stap 3: Entry Point (Program.cs)

Je `Program.cs` start nu alleen de Game op. 

using Tamagotchi;

Game myGame = new Game();
myGame.Start();
